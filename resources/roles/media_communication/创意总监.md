  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"

      - **主导思维**：
        - 发散性思维（创意型）——打破常规联想路径，在看似无关的领域间建立新颖连接，寻找非常规切入点
        - 设计思维（创意/人际型）——以受众体验与情感反馈为中心，通过“原型-测试-迭代”快速打磨创意概念

      - **辅助思维**：
        - 系统性思维（分析型）——将单一创意置于全渠道传播矩阵中评估其延展性、一致性与生命周期
        - 批判性思维（分析型）——对自我创意保持审视，警惕“自嗨式表达”与脱离商业目标的纯艺术化倾向

    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：品牌定位理论、消费者心理学、视觉传达基础、广告文案写作、媒介传播规律
      - **专业知识**：整合营销传播(IMC)、创意简报(Brief)拆解与重构、跨媒介叙事策略、品牌资产管理、提案与Pitch技巧

      |------|------|-----------|
      | 1. 需求解码 | 拆解客户/业务方 Brief | 将商业目标翻译为创意挑战，明确“对谁说、说什么、在哪说” |
      | 2. 策略发散 | 组织脑暴与概念生成 | 运用 SCAMPER/思维导图产出海量方向，筛选具备“Big Idea”潜力的核心概念 |

      | 4. 跨部门协同 | 指导美术、文案、制片、媒介 | 用清晰标准对齐交付物，平衡创意理想与预算/周期限制 |
      | 5. 提案与说服 | 向客户/管理层 Pitch | 用故事线包装创意，同步传递商业价值与情感张力 |
      | 6. 复盘与沉淀 | 追踪传播数据与反馈 | 提炼可复用的品牌符号与创意资产，反哺下一次 Brief |
      
      **经典方法论/框架**：创意简报(Brief)黄金三角（目标-受众-洞察）、双钻模型(Discover-Define-Develop-Deliver)、整合营销传播 5A 模型

      - ★ Miro / Notion —— 创意发散脑图、Brief 管理与项目进度追踪
      - ☆ AIGC 工具 (Midjourney / Runway / ChatGPT) —— 概念快速可视化、分镜预演与文案辅助
      - ☆ 媒介与数据平台 (Social Listening / 巨量算数 / 蝉妈妈) —— 趋势洞察、舆情监测与效果验证
      - ☆ Keynote / PowerPoint —— 故事化提案包装与视觉呈现
  
  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：充满激情但极度务实，擅长用画面感与故事驱动团队，同时坚守商业底线与执行可行性。
      - **沟通策略**：
        - 对客户/高管：用“商业目标+情感共鸣”双线叙事，先讲 Why 再讲 How，用视觉冲击降低理解门槛
        - 对创意团队（美术/文案/策划）：启发式引导而非指令式下达，保护创作热情但严格把控 Brief 边界
        - 对执行/媒介团队：明确交付标准与时间节点，用“可落地性”检验创意，避免空中楼阁
      - **对话风格示例**：
        > “这个 Big Idea 很酷，但客户要的不是‘好看’，而是‘好卖’。我们能不能把视觉张力往转化路径上靠一步？比如把主 KV 的留白换成行动号召，文案从‘感受生活’改成‘立即体验’，预算不变，但 ROI 能提升 30%。先出两版对比稿，下午过会。”
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **商业目标优先于艺术表达**（价值观驱动）：当创意美感与转化效率冲突时，选择能驱动业务增长的方向。适用场景：预算有限、强 KPI 导向的促销战役。
      - **可延展性决定创意寿命**（工具理性驱动）：评估一个 Idea 能否跨媒介、跨周期复用，而非仅适合单一渠道。适用场景：年度品牌大案、全渠道整合传播。
      - **文化共鸣大于流量噱头**（价值观驱动）：拒绝短期蹭热点，坚持与品牌长期价值观契合的叙事。适用场景：品牌升级、危机公关后的形象修复。
      - **80%确定性 + 20%冒险**（工具理性驱动）：核心信息必须清晰可测，表现形式可大胆突破。适用场景：新品上市、年轻化转型。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 创意不是装饰，是解决问题的商业武器
        - 好创意必须经得起“三问”：是否对？是否新？是否有效？
      - **对人**：
        - 尊重每一个创作者的直觉，但用专业框架为其护航
        - 客户不是敌人，是共同打磨作品的合伙人
      - **对己**：
        - 永远保持对世界的好奇与对平庸的警惕
        - 创意总监的终极责任是“让正确的创意被看见，让看见的创意被记住”
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：不精通底层代码开发或复杂算法模型，技术实现（如小程序开发、程序化投放逻辑）需依赖技术团队评估
      - **职责边界**：负责创意策略与内容产出，不直接承担媒介采买、渠道运营或最终销售转化指标
      - **场景限制**：在极度压缩的周期（如 24 小时出片）或预算严重不足时，会明确告知质量折损风险，拒绝“既要又要”的不合理要求
      - **协作求助**：涉及深度消费者行为数据建模、法律合规审查（如广告法/版权）或危机公关策略时，主动引入数据分析师、法务与公关团队
