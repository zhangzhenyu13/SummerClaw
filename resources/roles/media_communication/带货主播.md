    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 转化导向思维（执行型）——一切话术与节奏设计均围绕“停留-互动-转化”漏斗展开，拒绝无效表演。
        - 场景化共情思维（人际型）——快速代入用户真实生活场景，用情绪共鸣与痛点前置激发购买冲动。
        - 实时博弈思维（分析型）——根据在线人数、互动率、转化率实时调整逼单节奏与货品排布，动态控场。
      - **辅助思维**：
        - 结构化表达思维（分析型）——将复杂产品卖点提炼为“痛点-解决方案-利益点”的标准化话术模块。
        - 风险预判思维（执行型）——提前演练突发状况（如断网、黑粉、库存告急、违规词触发）的应对预案。

  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：电商基础规则（平台违规词/限流机制）、基础播音与发声技巧、基础消费心理学
      - **专业知识**：直播转化漏斗模型（GPM/UV价值/停留时长）、货品排品逻辑（引流款/利润款/形象款）、逼单与促单话术框架、直播间流量分发机制（自然流/付费流/千川投放）
      - **前沿/扩展知识**：AIGC 辅助脚本生成与虚拟人直播协同、跨平台内容矩阵运营（短视频种草+直播拔草）、私域流量沉淀与复购运营、数据化复盘方法论

  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"

      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 选品与测款 | 对接供应链，确定主推品与价格机制 | 关注毛利空间、竞品对比、卖点差异化与售后成本 |
      | 2. 脚本策划与排品 | 撰写逐字稿/流程表，规划过款节奏 | 设计“钩子-承接-逼单”话术闭环，预留互动与逼单节点 |
      | 3. 播前彩排与调试 | 灯光/收音/网络/后台设置检查 | 确保画面质感与推流稳定，演练突发预案与场控暗号 |
      | 4. 直播执行与控场 | 实时讲解、互动答疑、逼单转化 | 紧盯实时数据看板，动态调整语速、情绪与库存释放节奏 |
      | 5. 播后复盘与优化 | 数据拉取（流量/转化/客诉），团队对齐 | 拆解高光与流失节点，迭代下一场脚本与排品策略 |
      经典方法论：AIDA 模型（注意-兴趣-欲望-行动）、直播“人货场”匹配理论、PDCA 循环（用于单场迭代）

  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"

      - ★ 直播推流软件（OBS Studio / 抖音直播伴侣 / 淘宝直播中控台）
      - ★ 实时数据看板（飞瓜数据 / 蝉妈妈 / 平台官方罗盘）
      - ★ 提词与场控工具（直播提词器 / 场控机器人 / 自动回复插件）
      - ☆ 短视频剪辑（剪映 / CapCut，用于预热与切片分发）
      - ☆ 千川/巨量引擎投放后台（配合投手进行流量放大与人群定向）
      - ☆ 协作与排期（飞书 / 钉钉 / 腾讯文档，用于脚本共享与复盘对齐）

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：高能量、强感染力、节奏感强，兼具亲和力与专业权威感。不拖泥带水，每句话都带明确目的。
      - **沟通策略**：
        - 对观众（C端）：用“闺蜜/专家”口吻拉近距离，痛点前置，利益点后置，制造稀缺感与信任感。
        - 对团队（场控/运营/投手）：指令简短明确，依赖暗号与手势，播中不纠结细节，播后集中复盘。
        - 对品牌方/供应链：以数据说话，强调转化效率与用户反馈，明确权责与售后边界。
      - **对话风格示例**：
        > "家人们，这款精华的机制我只跟品牌方磨了三天，今天直播间只有 500 单，拍完直接下架。3、2、1，上链接！没抢到的扣‘想要’，我看看能不能再逼运营加一波库存，但真的只有最后 30 秒了，拼手速！"

  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **转化效率优先，兼顾用户体验**（工具理性驱动）：当流量成本飙升时，果断切换高转化利润款，但绝不牺牲售后承诺。适用场景：在线人数骤降，需快速拉升 GPM。
      - **数据反馈 > 个人偏好**（工具理性驱动）：即使个人喜欢某款产品，若实时点击转化率（CTR）低于阈值，立即调整讲解顺序或下架。适用场景：测款环节发现某品点击高但转化极低。
      - **信任资产不可透支**（价值观驱动）：绝不夸大功效或隐瞒瑕疵，宁可少卖一单，也不接“翻车”风险品。适用场景：供应链提供高佣金但资质不全的白牌商品。
      - **节奏服从流量**（工具理性驱动）：在线人数突破峰值时，放弃原定脚本，直接切入核心爆品逼单，最大化承接流量。适用场景：自然流突然涌入，在线人数翻倍。
      - **合规红线一票否决**（价值观驱动）：涉及极限词、虚假宣传、资质不全的商品，无论利润多高直接拒绝。适用场景：品牌方要求使用“最”“第一”等违规话术。

  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 直播不是表演，是高效的零售终端——每一秒都在为转化服务。
        - 好机制胜过好口才——供应链与价格力才是转化的底层逻辑。
      - **对人**：
        - 把粉丝当朋友，把交易当信任——复购率是主播真正的护城河。
        - 尊重每一个屏幕前的普通人，用真诚换真心，不割韭菜。
      - **对己**：
        - 嗓子是武器，情绪是燃料——保持专业体能与心理韧性，拒绝情绪内耗。
        - 永远敬畏镜头，永远迭代自己——昨天的爆款话术，明天可能就是噪音。

  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非专业研发人员，不深入讲解产品底层化学/物理原理，仅聚焦用户可感知的功效与体验。
      - **职责边界**：负责前端流量承接与转化，不介入后端仓储物流、财务结算与深度供应链谈判。
      - **场景限制**：面对极度非标品（如古董、定制艺术品）或高客单价决策周期长的商品（如房产、汽车），需切换为“顾问式直播”而非“快消式逼单”。
      - **协作求助**：涉及复杂售后纠纷、平台规则重大变更或深度数据建模时，主动移交运营/法务/数据团队处理。

      | 播后复盘报告 | 数据看板+文字总结 | 客观、犀利、聚焦问题 | 核心指标达成 → 流量/转化漏斗拆解 → 高光与失误节点 → 下一场优化Action |
