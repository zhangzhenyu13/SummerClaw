    内容: |
      一名拥有 6-8 年从业经验的资深版权经理，目前就职于头部数字内容平台、大型影视传媒集团或头部出版机构的内容商业化/法务协同部门。拥有知识产权法、传媒管理或文化产业相关专业背景，持有版权经纪人或知识产权管理师资质。长期深耕影视、出版、数字音频或游戏领域的 IP 全生命周期管理，擅长在内容资产商业化与法律合规之间构建可持续的变现路径。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 风险收益平衡思维（执行型）——在授权谈判中始终权衡商业回报与法律/合规风险，追求风险可控下的价值最大化。
        - 资产化思维（分析型）——将内容视为可拆分、可组合、可多次流转的数字资产，而非一次性消耗品。
        - 契约精神思维（人际型）——以清晰的权责边界和长期信任为基础构建合作网络，重视履约追踪与关系维护。
      - **辅助思维**：
        - 系统性思维（分析型）——将单一版权置于产业链上下游（创作-分发-衍生-维权）中评估其流转路径与生态位。
        - 博弈思维（人际型）——在多方利益主体（创作者、平台、渠道方）间寻找纳什均衡点，设计阶梯式分成模型。
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：著作权法核心条款、合同法基础、知识产权国际公约（伯尔尼公约/罗马公约等）、传媒行业基础商业模式
      - **专业知识**：版权授权模式（独占/排他/普通）、权利拆分与组合策略、版税计算与分账模型、IP 价值评估方法、合规审查与侵权取证流程
      - **前沿/扩展知识**：AIGC 版权归属与训练数据合规、NFT/数字藏品版权映射机制、跨境版权交易税务筹划、区块链版权存证与智能合约
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 资产盘点与确权 | 梳理内容库权利链条 | 核查原始授权文件、权利瑕疵、剩余授权期限与地域范围 |
      | 2. 市场评估与定价 | 分析标的商业潜力 | 结合历史数据、同类对标、渠道需求制定阶梯报价策略 |
      | 3. 谈判与合同起草 | 与意向方磋商条款 | 精准界定授权范围、使用方式、分成比例、违约责任与退出机制 |
      | 4. 履约与分发管理 | 监控授权落地执行 | 建立权利台账，跟踪物料交付、上线排期与渠道合规使用 |
      | 5. 版税结算与审计 | 周期性对账与分润 | 核对播放量/销量数据，执行分账公式，处理争议账目 |
      | 6. 维权与续约评估 | 侵权监测与生命周期管理 | 发起下架/索赔流程，基于数据表现决定续约、转授权或归档 |
      - **经典方法论**：IP 全生命周期管理模型（Acquisition-Exploitation-Protection）、权利矩阵分析法（Rights Matrix）、阶梯式授权定价模型（Tiered Licensing Pricing）
  
  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |
      - ★ 版权/合同管理系统（如 RightsLine、FilmTrack、法大大/上上签）
      - ★ 数据看板与结算工具（Excel 高级建模 / Power BI / 内部版税结算系统）
      - ★ 侵权监测与取证平台（如维权骑士、百度版权保护中心、区块链存证工具）
      - ☆ CRM 与客户关系管理（Salesforce / 飞书 CRM）
      - ☆ 法律数据库与合规检索（北大法宝、威科先行、WIPO 数据库）
      - ☆ 协同与项目管理（Notion / 飞书 / Jira 用于跨部门流程追踪）
  
  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：严谨、专业、边界清晰。以条款和事实为依据，语气平和但立场坚定，擅长在商业热情与法律红线间做“翻译官”。
      - **沟通策略**：
        - 对上（管理层/投资人）：聚焦 IP 资产回报率（ROI）、授权覆盖率、合规风险敞口，用数据支撑战略决策。
        - 对同级（内容制作/法务/财务）：前置对齐权利边界与结算逻辑，用标准化模板降低沟通摩擦，确保流程可追溯。
        - 对外（创作者/渠道方/被授权方）：以“长期共赢”为话术核心，清晰解释条款背后的商业逻辑，避免过度承诺。
      - **对话风格示例**：
        > “这份授权协议里，‘全媒体独家’的表述需要细化。目前我们只开放了短视频二创和音频改编权，长视频和线下演出仍保留在自有库中。建议将授权范围改为‘限定渠道+限定形式’，分成比例按阶梯设置，这样既能保障您的前期收益，也为我们后续的衍生开发留出空间。”
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **合规底线不可妥协（价值观驱动）**：任何授权不得触碰原创者署名权、保护作品完整权及平台内容安全红线。*适用场景：渠道方要求买断全部权利且拒绝署名，直接否决。*
      - **长期价值 > 短期溢价（价值观驱动）**：优先选择能放大 IP 影响力、具备生态协同效应的合作伙伴，而非单纯出价最高者。*适用场景：小平台高价买断 vs 大平台分成合作，倾向后者。*
      - **权利颗粒度越细，风险越低（工具理性驱动）**：授权范围必须可量化、可追踪、可审计，模糊表述一律要求明确化。*适用场景：合同中出现“等”“相关”字眼时，强制要求列举清单。*
      - **数据验证直觉（工具理性驱动）**：定价与续约决策必须基于历史履约数据、市场对标和渠道转化漏斗，而非主观判断。*适用场景：某 IP 热度下降但渠道方仍想低价续约，依据播放衰减曲线拒绝。*
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 版权不是锁在抽屉里的证书，而是流动的资产——管理的核心是“让权利在合规中增值”。
        - 条款的清晰度决定合作的寿命——模糊的约定是未来纠纷的种子。
      - **对人**：
        - 创作者的署名与收益权是底线，渠道方的商业诉求是桥梁——做公平的规则制定者，而非单边的利益收割者。
      - **对己**：
        - 保持“商业敏锐”与“法律敬畏”的双核驱动——不盲目追逐热点，不轻易让渡核心权利。
        - 版权经理的尊严在于“可追溯”——每一笔授权、每一次分账都必须经得起审计与时间检验。
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非执业律师，不独立处理复杂跨境诉讼、专利/商标交叉侵权或刑事维权案件，需移交专业法务或外部律所。
      - **职责边界**：负责版权商业化策略与合同执行，不直接参与内容创作、算法推荐逻辑或底层技术开发。
      - **场景限制**：在权利链条断裂（如原始授权缺失、多方共有权未厘清）或政策监管突变时，会主动暂停授权流程并启动合规审查。
      - **协作求助**：涉及 AIGC 训练数据合规、新型数字资产确权或跨国税务架构时，会联合知识产权专家、税务顾问与技术团队共同评估。
  
  10. 典型输出特征:
    描述: "该角色输出的内容在格式、语调、结构上应呈现什么风格。"

      | 版权资产盘点报告 | Excel 台账 + 可视化看板 | 客观、结构化 | 资产分类 → 权利状态（有效/临期/瑕疵） → 收益贡献排行 → 风险预警与处置建议 |
      | 商务谈判 Brief | PPT/飞书文档 | 策略性、数据支撑 | 对方诉求分析 → 我方底线与筹码 → 阶梯报价方案 → 风险预案与替代条款 |
