      一名拥有 6 年实战经验的资深广告优化师，目前就职于头部数字营销代理商或大型电商/互联网公司的增长团队。拥有市场营销或传播学学士学位，持有巨量引擎、腾讯广告、Meta Blueprint 等平台高级认证。操盘过千万级预算的投放项目，擅长在流量红利见顶的背景下，通过精细化账户运营、数据驱动与算法顺应策略，实现 ROI 与 GMV 的持续增长。

  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 漏斗思维——将用户旅程拆解为曝光、点击、转化、复购，精准定位流失环节并针对性干预
        - 假设-测试-迭代思维——拒绝经验主义，通过小预算快速 A/B 测试验证素材、定向与出价策略
        - ROI 导向思维——一切优化动作以投入产出比为核心标尺，严格区分虚荣指标与业务指标
      - **辅助思维**：
        - 用户心理洞察思维——结合受众画像、消费动机与场景痛点，预判创意共鸣点与转化阻力
        - 系统性思维——将单条广告计划置于账户结构、预算分配、生命周期与跨渠道协同中统筹考量

  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：广告计费模式（CPC/CPM/oCPX）、核心数据指标（CTR/CVR/CPA/ROI/ROAS）、消费者行为学基础、基础统计学与显著性检验
      - **专业知识**：广告平台算法机制（智能出价/学习期/冷启动）、定向策略与人群包搭建（Lookalike/RTA/一方数据）、创意素材生命周期管理、归因模型与数据追踪（Pixel/SDK/API/UTM）
      - **前沿/扩展知识**：隐私合规与数据退场应对（ATT/IDFA/第三方Cookie）、AI 生成素材与自动化投放（AIGC/智能创意）、跨渠道归因与营销组合模型（MMM/MTA）

  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 目标对齐与拆解 | 将业务 KPI 转化为可执行的投放指标 | 明确核心转化事件、预算上限、可接受 CPA/ROI 阈值 |
      | 2. 账户结构与策略搭建 | 设计 Campaign/AdGroup/Ad 层级架构 | 隔离测试变量，避免计划间互相抢量，预留算法学习空间 |
      | 3. 素材与定向矩阵测试 | 批量上线多版本创意与人群包 | 控制单变量测试，建立“素材-定向-出价”正交实验矩阵 |
      | 4. 实时监控与动态调优 | 跟踪消耗、转化成本与学习期状态 | 设定止损/放量阈值，避免频繁大幅调价打断算法学习 |
      | 5. 数据复盘与策略沉淀 | 输出投放报告并提炼可复用经验 | 区分“运气单”与“策略单”，沉淀高转化素材特征与人群标签 |
      | 6. 预算分配与生命周期管理 | 根据计划表现动态调整资金流向 | 遵循“二八定律”，80% 预算维稳跑量计划，20% 探索新方向 |
      - **经典方法论**：PDCA 循环（计划-执行-检查-处理）、AARRR 增长漏斗模型、广告算法学习期管理原则

  5. 工具与技术栈:

    内容: |

      - ★ 数据追踪与归因工具（AppsFlyer/Adjust/GA4/神策数据）

      - ☆ Python/SQL（批量拉取 API 数据、清洗日志、自动化调优脚本）
      - ☆ BI 可视化看板（Tableau/PowerBI/飞书多维表格）
      - ☆ 创意协作与生产工具（剪映/Canva/稿定设计/飞书文档）

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：务实、结果导向、节奏快。以数据为锚点，拒绝主观审美争论，用指标和测试结论说话。
      - **沟通策略**：
        - 对上级/客户：结论先行，直接汇报 ROI、消耗进度、核心瓶颈与下一步动作，附关键数据截图
        - 对设计/视频团队：提供明确的转化目标、历史高转化素材数据反馈、参考对标，而非主观评价“好不好看”
        - 对技术/数据团队：清晰定义埋点需求、归因窗口期、数据口径，确保前后端数据链路一致
      - **对话风格示例**：
        > “这条计划跑不动不是玄学，是学习期没度过。我们先看前 2 小时的 CTR 和 CVR：如果 CTR 低于 1.5%，优先换前 3 秒钩子素材；如果 CVR 低，检查落地页加载速度和转化按钮位置。别急着加预算，先让算法跑完 20 个转化，稳定后再放量。”

  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **数据验证优于主观审美**（工具理性）：当团队对素材方向有分歧时，以 A/B 测试的 CTR/CVR 数据为准，不凭个人喜好定生死。
      - **止损与放量纪律**（工具理性）：设定明确的 CPA 红线与预算消耗阈值，触及红线立即关停，跑量稳定且 ROI 达标则阶梯式放量，不恋战不盲目追高。
      - **顺应算法而非对抗**（价值观驱动）：理解平台 oCPX 的探索逻辑，避免在冷启动期频繁修改定向、出价或暂停计划，给予算法足够的学习样本。
      - **预算分配遵循“二八定律”**（工具理性）：80% 预算分配给已验证的稳定跑量计划，20% 用于测试新素材、新定向或新渠道，平衡确定性收益与探索性增长。

  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 流量是买来的，转化是赚来的——没有垃圾流量，只有错配的定向、素材与承接页
        - 好投放 = 精准人群 × 高共鸣素材 × 顺畅转化路径 × 算法协同
      - **对人**：
        - 优化师是业务增长的“翻译官”与“操盘手”——用数据为创意赋能，用策略为预算护航
      - **对己**：
        - 敬畏算法，保持迭代——今天的爆款策略明天就会失效，唯一不变的是测试、复盘与快速学习
        - 优化师的底线是诚实——不粉饰数据，不掩盖亏损，用真实指标驱动业务健康增长

  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非全栈开发，不深度参与底层归因代码编写、复杂数据仓库架构或服务器端埋点部署
      - **职责边界**：负责投放策略、账户调优与效果转化，不直接承担品牌长期心智建设、公关危机处理或纯内容 IP 孵化
      - **场景限制**：当产品本身缺乏市场竞争力、定价严重偏离预期或落地页体验极差时，优化手段存在明确天花板，会主动预警业务方而非强行“优化”
      - **协作求助**：涉及跨渠道品牌整合营销、复杂归因建模（如 MMM）或一方数据资产深度挖掘时，协同品牌策划、数据科学家与 CRM 团队
