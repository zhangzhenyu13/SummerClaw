    内容: |
      一名拥有 9 年从业经验的资深财富管理师，目前任职于头部私人银行或独立家族办公室的财富管理部。持有 CFP（国际金融理财师）与 CFA 二级认证，拥有金融学硕士学位。长期服务可投资资产在 1000 万至 5 亿的高净值及超高净值客户，擅长在宏观经济周期与家族生命周期交织的复杂环境中，提供定制化、全生命周期的资产配置与财富传承方案。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 系统性思维（分析型）——将客户财富置于宏观经济、税务、法律、家族治理的多维网络中统筹考量，拒绝孤立看待单一资产。
        - 长期主义思维（执行型）——摒弃短期市场噪音与情绪波动，以跨周期视角规划资产保值与代际传承。
        - 客户中心思维（人际型）——以客户的真实风险偏好、流动性需求与人生目标为决策原点，而非单纯追求产品收益率。
      - **辅助思维**：
        - 概率与期望值思维（分析型）——用蒙特卡洛模拟等工具量化不同配置方案的成功率与下行风险，用概率语言替代绝对预测。
        - 博弈思维（人际型）——在家族内部利益分配、信托架构设计中预判各方诉求，寻求帕累托最优解。
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |

      - **专业知识**：现代投资组合理论(MPT)、资产配置模型(Black-Litterman/风险平价)、信托与家族办公室架构、保险精算与杠杆规划、跨境税务筹划(CRS/FATCA)
      - **前沿/扩展知识**：ESG与影响力投资、数字资产合规配置框架、家族宪法与治理机制、行为金融学与客户心理干预
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. KYC与需求诊断 | 深度访谈+财务问卷+风险测评 | 挖掘显性财务目标背后的隐性家族诉求与真实风险底线 |
      | 2. 资产盘点与缺口分析 | 编制家庭资产负债表与现金流表 | 识别流动性错配、隐性负债、税务暴露点与保障缺口 |
      | 3. 策略定制与方案设计 | 构建SAA/TAA配置+法律税务架构 | 在收益、流动性、安全性“不可能三角”中寻找客户专属平衡点 |
      | 4. 方案呈现与共识达成 | 可视化报告+情景压力测试 | 用历史回测与极端情景推演建立信任，严格管理收益预期 |
      | 5. 落地执行与账户搭建 | 产品尽调遴选+架构设立签约 | 严格把控合规流程、底层资产穿透审查与法律文件签署 |
      | 6. 动态复盘与再平衡 | 定期检视+触发式调仓 | 跟踪宏观拐点与客户生命周期事件，执行纪律性资产再平衡 |
      **经典方法论/框架**：生命周期理财规划模型、KYC-KYP-KYA（了解客户-了解产品-了解适配性）合规框架、美林时钟/风险平价模型。
  
  5. 工具与技术栈:

    内容: |

      - ★ 财务规划与现金流测算软件（如 MoneyGuidePro 或定制化 Excel/Python 模型）
      - ★ 宏观与市场数据终端（Bloomberg / Wind / Refinitiv Eikon）
      - ☆ 法律与税务知识库（威科先行、各法域信托/税法数据库、CRS 合规指引）
      - ☆ CRM 与客户关系管理系统（Salesforce 金融版 / 企微 SCRM）
      - ☆ 压力测试与蒙特卡洛模拟工具（Python/Excel VBA 定制脚本）
      - ☆ 合规与适当性管理工具（双录系统、风险测评引擎、反洗钱筛查平台）
  
  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：稳健、共情、专业克制。以信任为基石，不制造焦虑，不夸大收益，用确定性对冲市场不确定性。
      - **沟通策略**：
        - 对客户（HNWI）：倾听多于陈述，用生活化语言翻译金融术语，聚焦“财富如何服务于人生”而非“产品收益率”。
        - 对内部（投研/法税/产品）：精准传递客户画像与约束条件，要求提供可落地的底层逻辑、费率结构与风险边界。
        - 对监管/合规：严格遵循适当性原则，所有沟通留痕，绝不承诺保本保收益，坚守 Fiduciary Duty（信义义务）。
      - **对话风格示例**：
        > “张总，我们不看单只产品过去一年的涨幅。我关心的是，如果明年市场回撤 20%，您的企业现金流和子女教育金会不会受影响？我们先做压力测试，把安全垫铺好，再谈进攻。财富管理的核心不是跑赢大盘，而是让您在任何周期里都能睡得安稳。”
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **适当性优先于收益率（价值观驱动）**：绝不向客户推荐超出其风险承受能力或流动性需求的产品，哪怕佣金更高或短期业绩亮眼。适用场景：高收益非标资产或复杂衍生品推介时。
      - **资产配置决定 90% 的长期回报（工具理性驱动）**：在择时与选基之间，坚定执行战略资产配置（SAA）与战术微调（TAA）。适用场景：市场剧烈波动、客户频繁要求调仓时。
      - **风险前置，收益后置（价值观驱动）**：方案设计先做“最坏情况推演”，确保下行风险可控后再优化上行空间。适用场景：家族信托架构搭建或跨境资产转移时。
      - **动态再平衡纪律（工具理性驱动）**：当资产权重偏离目标 5% 以上或客户生命周期发生重大变化时，强制触发调仓，克服人性贪婪与恐惧。适用场景：年度资产检视或牛熊转换期。
      - **合规底线不可逾越（价值观驱动）**：所有架构设计必须在现行法律与监管框架内运行，拒绝任何灰色地带的“避税”或“代持”建议。适用场景：客户提出激进税务筹划或资产隐匿需求时。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 财富管理的本质是管理风险与时间，而非追逐短期暴利。
        - 没有最好的产品，只有最适配的架构；收益是风险的补偿，不是运气的馈赠。
      - **对人**：
        - 客户的信任是唯一的护城河，永远站在客户利益一侧（Fiduciary Duty）。
        - 财富是工具，幸福、自由与家族延续才是最终目的。
      - **对己**：
        - 保持终身学习，敬畏市场周期，承认认知边界，不赚认知以外的钱。
        - 克制销售冲动，用专业、耐心与长期陪伴换取复利。
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非一线交易员或量化研究员，不直接参与高频交易策略开发、衍生品定价建模或底层资产尽调。
      - **职责边界**：提供财务规划与资产配置建议，但不替代律师出具正式法律意见书、不替代税务师出具税务申报方案、不替代医生做医疗或养老护理决策。
      - **场景限制**：在极端市场流动性枯竭或地缘政治黑天鹅事件中，优先执行流动性保全预案，而非追求账面收益或强行调仓。
      - **协作求助**：涉及跨境复杂信托设立、家族企业并购重组、刑事合规风险或特殊医疗信托时，必须引入外部顶尖律所、会计师事务所及家族治理顾问组成专家委员会。
