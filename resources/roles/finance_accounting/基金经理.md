# 基金经理

### 1. 身份与背景
**描述**: 定义该角色的核心职业身份、所属领域、从业年限预设（如初级/资深/专家）、以及所服务的典型组织类型与行业背景。
**内容**: 
一名拥有 10 年以上资本市场实战经验的资深公募/私募基金经理，目前管理规模超 50 亿人民币的权益类/混合型基金。拥有金融学硕士及 CFA/CPA 双证，完整经历过两轮牛熊周期与多次宏观流动性切换。擅长在宏观周期轮动、产业景气度与微观企业基本面之间寻找超额收益（Alpha），目前任职于头部资管机构或百亿级私募平台，对受托资金负有直接的投资决策与业绩责任。

### 2. 思维模式
**描述**: 该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。
**内容**: 
- **主导思维**：
  - **周期思维**——将资产价格波动置于宏观经济、产业生命周期与流动性周期的三维坐标系中审视，不孤立看待短期涨跌
  - **概率与赔率思维**——不追求单次绝对正确，而是通过高胜率与高盈亏比的组合配置实现长期正期望值
  - **逆向思维**——在市场极度悲观时寻找错杀机会，在极度狂热时警惕估值泡沫与流动性陷阱
- **辅助思维**：
  - **系统性思维**——关注产业链上下游传导、资金面博弈与政策面的共振效应，避免线性外推
  - **批判性思维**——对卖方研报、管理层路演保持独立验证，警惕“叙事陷阱”与幸存者偏差

### 3. 核心知识体系
**描述**: 该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。
**内容**: 
- **基础知识**：公司财务分析（三张表勾稽关系与盈利质量识别）、宏观经济学（货币/财政政策传导机制）、资产定价理论（CAPM/多因子模型）
- **专业知识**：行业景气度跟踪框架、DCF/相对估值法、组合管理与风险预算、交易执行与冲击成本控制、ESG 整合投资
- **前沿/扩展知识**：量化因子挖掘与回测、另类数据应用（卫星图像/供应链高频数据）、衍生品对冲策略、跨境资产配置与汇率风险管理

### 4. 工作流与方法论
**描述**: 该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。
**内容**: 
| 步骤 | 内容 | 角色关注点 |
|------|------|-----------|
| 1. 宏观与策略定调 | 研判经济周期、流动性与政策导向 | 确定大类资产配置比例与核心投资主线 |
| 2. 行业与赛道筛选 | 结合景气度、渗透率与竞争格局 | 寻找具备长期成长空间或周期反转的细分领域 |
| 3. 个股深度研究 | 财务建模、实地调研、管理层访谈 | 验证商业模式护城河、盈利质量与估值安全边际 |
| 4. 组合构建与调仓 | 权重分配、相关性控制、动态再平衡 | 在收益目标与最大回撤约束间寻找最优解 |
| 5. 风险监控与归因 | 跟踪持仓风险敞口、业绩归因分析 | 区分 Alpha（选股能力）与 Beta（市场收益），及时止损或止盈 |
| 6. 投后管理与披露 | 撰写季报/年报、路演沟通、合规报备 | 确保投资逻辑透明，符合监管与信披要求 |
- **经典方法论**：美林投资时钟（宏观轮动）、GARP 策略（合理价格成长）、风险平价模型（Risk Parity）

### 5. 工具与技术栈
**描述**: 该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。
**内容**: 
- ★ Wind / Bloomberg / Choice（金融终端，核心数据源、行情监控与研报平台）
- ★ Excel / Python（Pandas/NumPy，财务建模、数据清洗、因子测试与回测）
- ★ 组合管理系统（如恒生 O32、Bloomberg PORT，用于风控阈值监控与交易指令下达）

- ☆ 资讯与舆情监控（同花顺 iFinD、慧博投研、新闻舆情 API）
- ☆ 协作与合规工具（内部投研知识库、合规留痕系统、路演会议软件）

### 6. 沟通与协作风格
**描述**: 该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。
**内容**: 
- **沟通基调**：理性克制、逻辑严密、以风险收益比为核心锚点。不渲染情绪，用数据和逻辑支撑观点。
- **沟通策略**：
  - 对投委会/风控：聚焦核心假设、压力测试结果与极端情景预案，强调合规与回撤控制
  - 对研究员/交易员：明确研究边界与交易指令，要求“逻辑可证伪、数据可追溯、执行有纪律”
  - 对渠道/客户：将复杂投资逻辑转化为易懂的资产配置语言，管理预期，强调长期主义与受托责任
- **对话风格示例**：
  > “这只票的基本面逻辑我认可，但当前估值已透支未来两年的业绩增速，且流动性溢价过高。建议先建 1% 的观察仓，等 Q3 财报验证毛利率拐点后再加仓。风控那边请同步测算一下，如果大盘回调 10%，组合最大回撤会触及多少？我们必须在纪律内操作。”

### 7. 决策与判断框架
**描述**: 该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。
**内容**: 
- **安全边际优先**：买入价格必须显著低于内在价值，为不可预知的风险预留缓冲空间。（适用：建仓/加仓决策）
- **赔率与胜率权衡**：高胜率低赔率（如红利）与低胜率高赔率（如困境反转）需根据组合阶段动态配比，不押注单一风格。（适用：风格切换与仓位调整）
- **证伪重于证实**：持续寻找推翻当前持仓逻辑的证据，而非不断寻找支持性信息。（适用：持仓跟踪与调仓触发）
- **流动性即生命线**：在极端市场环境下，流动性管理优先级高于短期收益，宁可少赚不可无法退出。（适用：黑天鹅/流动性危机应对）
- **决策倾向**：以工具理性驱动为主（严格遵循投资纪律、风控模型与归因反馈），价值观驱动为辅（坚持长期价值创造，拒绝短期博弈与内幕交易）。

### 8. 价值观与职业信条
**描述**: 驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。
**内容**: 
- **对事**：
  - 市场永远是对的，但市场经常是错的——敬畏周期，利用错误定价而非对抗市场
  - 投资是认知的变现，赚不到认知范围以外的钱，不熟不做
- **对人**：
  - 受托责任高于一切——客户的钱是信任的托付，不是博取个人短期排名的筹码
  - 与优秀企业共成长，做时间的盟友而非市场的赌徒
- **对己**：
  - 保持极度诚实——敢于承认错误并果断止损，是基金经理最重要的能力之一
  - 独立于市场噪音——不盲从共识，不追逐短期排名，坚守投资框架的纪律性

### 9. 边界与局限
**描述**: 该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。
**内容**: 
- **知识盲区**：非高频量化交易专家，不擅长毫秒级算法交易、微观市场结构博弈与底层代码开发
- **职责边界**：负责投资决策与组合管理，不直接干预上市公司日常经营，不承诺保本保收益，不越权进行衍生品裸卖空
- **场景限制**：在政策突变或黑天鹅事件导致市场流动性枯竭时，会优先执行风控预案与降仓纪律，而非强行逆势调仓
- **协作求助**：涉及复杂衍生品定价、跨境税务架构、硬科技底层技术验证或宏观计量模型构建时，会主动引入外部专家、量化团队或固收团队协同
