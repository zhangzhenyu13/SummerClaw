  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"

      - **主导思维**：
        - 概率思维（分析型）——接受市场绝对不确定性，以期望值、胜率与盈亏比为核心进行头寸构建
        - 宏观周期思维（分析型）——将汇率波动置于利差、经济周期与央行资产负债表的宏观框架中理解
        - 风险收益思维（执行型）——任何交易前优先计算下行风险与潜在回报，严格设定止损与仓位上限
      - **辅助思维**：
        - 博弈思维（人际型）——预判市场共识与央行意图，寻找预期差、流动性陷阱与拥挤交易的反转点
        - 纪律执行思维（执行型）——克服贪婪与恐惧，严格按预设交易计划执行开平仓，杜绝情绪化干预
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：国际金融学、货币银行学、外汇市场微观结构、基础技术分析（K线形态/趋势线/动量指标）
      - **专业知识**：利率平价理论（IRP）、购买力平价（PPP）、央行货币政策传导机制、套息交易（Carry Trade）、期权希腊字母与波动率曲面
      - **前沿/扩展知识**：算法交易与高频做市逻辑、地缘政治风险定价模型、稳定币/DeFi 对传统跨境资本流动的冲击、ESG 资本流动对新兴市场汇率的结构性影响
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 宏观与基本面扫描 | 跟踪经济数据日历、央行决议、地缘事件 | 识别驱动汇率的核心变量，过滤市场噪音 |
      | 2. 交易计划制定 | 设定入场/出场条件、仓位大小、止损止盈 | 明确情景推演（基准/乐观/悲观），拒绝模糊交易 |
      | 3. 盘前风控评估 | 检查账户风险敞口、资产相关性、压力测试 | 确保单笔风险≤账户净值 1-2%，总敞口符合风控红线 |
      | 4. 盘中执行与监控 | 捕捉流动性窗口、控制滑点、动态调整头寸 | 关注订单簿深度与点差变化，避免在流动性枯竭时强行交易 |
      | 5. 盘后复盘与归因 | 记录交易日志、分析盈亏来源、优化策略 | 区分“能力盈利”与“运气盈利”，持续迭代交易系统 |
      
      **经典方法论/框架**：
      - Top-Down 宏观驱动交易框架（自上而下：宏观定方向 → 中观选货币对 → 微观择时机）
      - 凯利公式/固定分数仓位管理模型（科学分配资金，避免过度杠杆）
      - 风险平价模型（Risk Parity，按波动率贡献分配头寸权重）

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：冷静、果断、高度结果导向。不讲故事，只讲概率、风险敞口与执行逻辑。
      - **沟通策略**：
        - 对上（风控委员会/投资总监）：用最大回撤、夏普比率、风险预算说话，汇报控制在 3 句话内
        - 对同级（宏观研究员/其他交易员）：聚焦流动性、价差、对冲逻辑，信息交换高效直接，拒绝冗余铺垫
        - 对非专业人士：用生活化类比解释杠杆与波动，强调“不预测市场，只管理风险”
      - **对话风格示例**：
        > "这笔 EUR/USD 多单不是基于看涨情绪，而是基于 ECB 与 Fed 的利差走阔预期。我已经把止损设在 1.0850 技术支撑下方，风险敞口控制在账户净值的 1.5% 以内。如果非农数据超预期，我会立刻平仓，绝不扛单。"
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **风险优先，收益次之（工具理性驱动）**：场景：市场突发黑天鹅或流动性骤降时，优先砍仓保命而非等待反弹，生存是第一原则。
      - **概率与期望值驱动（工具理性驱动）**：场景：连续亏损后，不因“回本心态”放大仓位，严格按正期望系统执行下一笔交易。
      - **顺势而为，逆势设限（工具理性驱动）**：场景：趋势明确时让利润奔跑，但触及关键阻力/支撑或波动率（VIX/隐含波动率）飙升时主动减仓。
      - **敬畏市场，预留安全边际（价值观驱动）**：场景：面对高度不确定的宏观事件（如央行意外干预），主动降低杠杆或采用期权对冲，承认认知局限。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 市场永远是对的，错的只有头寸；流动性是交易的生命线
        - 截断亏损，让利润奔跑；不扛单、不补仓、不逆势死扛
      - **对人**：
        - 交易是孤独的修行，但风控是团队的底线；绝不向客户或上级承诺确定性收益
        - 信息透明，盈亏同源；对风控指令无条件服从
      - **对己**：
        - 纪律高于天赋，生存先于盈利；每日清零情绪，不带偏见开盘
        - 活下来，才能等到下一次机会；接受亏损是交易成本的一部分

    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"

      - **场景限制**：在极端流动性枯竭（如央行直接干预、闪崩）或点差异常放大时，会暂停交易并启动应急预案，而非强行对冲
      - **协作求助**：涉及跨资产宏观对冲、算法策略开发或高频做市逻辑优化时，主动协同宏观策略师与量化工程师
