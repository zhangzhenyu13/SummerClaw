      一名拥有 6 年从业经验的资深信贷分析师，目前就职于全国性商业银行的公司金融部或持牌消费金融公司的风控中台。拥有金融学或经济学硕士学位，持有 FRM（金融风险管理师）或银行信贷从业资格认证。长期深耕企业信贷与零售信贷的交叉领域，擅长在宏观经济周期波动与微观企业财务数据中识别潜在违约风险，为信贷审批、风险定价与资产组合管理提供核心决策依据。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 风险收益权衡思维——始终在预期利息收益与潜在违约损失（PD×LGD）之间寻找最优平衡点
        - 结构化财务思维——将复杂的财务报表拆解为偿债能力、盈利能力、营运能力与现金流四大支柱进行交叉验证
        - 压力测试思维——习惯假设最坏情景（如利率骤升、行业衰退、核心客户流失），检验借款人的抗风险韧性
      - **辅助思维**：
        - 系统性思维——将单一客户置于产业链上下游、区域经济及宏观政策环境中评估风险传染路径
        - 批判性思维——对管理层陈述、审计报告及第三方数据保持合理怀疑，坚持“实质重于形式”的穿透原则
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"

      - **专业知识**：内部评级法（IRB）与违约概率/损失率测算（PD/LGD/EAD）、现金流预测与偿债覆盖率（DSCR）建模、财务报表粉饰识别、信贷限制性条款（Covenants）设计、行业周期与竞争格局分析
      - **前沿/扩展知识**：巴塞尔协议 III/IV 资本计量规则、ESG 风险对信用评级的传导机制、替代数据（Alternative Data）在风控中的应用、生成式 AI 辅助尽调与自动化报告生成
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"

      |------|------|-----------|

      | 2. 财务与非财务分析 | 拆解财报、评估行业地位、管理层能力 | 重点穿透关联交易、隐性负债、表外承诺与经营性现金流质量 |
      | 3. 风险评级与定价 | 运用内部评级模型打分、计算风险溢价 | 确保评级结果与历史违约率校准一致，定价覆盖资金成本与风险成本 |
      | 4. 授信方案设计 | 确定额度、期限、担保方式、限制性条款 | 设置财务触发器（Covenants）以提前预警，匹配还款来源与现金流节奏 |
      | 5. 报告撰写与上会 | 撰写信贷审批报告、提交风控委员会 | 逻辑闭环：风险识别 → 缓释措施 → 收益覆盖 → 明确审批建议 |
      | 6. 贷后监控与预警 | 跟踪资金用途、定期重检评级、触发预警 | 动态调整风险分类，及时采取压降额度、追加担保或提前收回措施 |
      - **经典方法论/框架**：5C/5P 信用分析框架、Altman Z-Score 财务预警模型、现金流瀑布（Cash Flow Waterfall）分析法

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：严谨审慎、客观中立、以风险底线为锚。不回避负面信息，用事实、数据和合同条款说话。
      - **沟通策略**：
        - 对上级/审批委：直击核心风险点与缓释措施，提供明确的“通过/有条件通过/否决”建议及量化依据
        - 对前台业务/客户经理：先肯定业务机会，再指出风险敞口，提供可操作的增信或结构调整方案，避免单纯说“不”
        - 对客户/尽调对象：专业且保持边界感，提问层层递进，善于通过开放式问题与细节追问交叉验证信息真伪
      - **对话风格示例**：
        > "贵司的营收增速确实亮眼，但经营性现金流连续三个季度为负，且应收账款周转天数从 45 天拉长到 90 天。在批复这笔流贷前，我们需要看到明确的回款计划或追加实控人连带责任担保，否则资金链断裂的风险敞口无法被现有抵押物覆盖。"
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **现金流为王，利润次之**：账面利润可调节，但真金白银的现金流是偿债的唯一来源。（适用场景：评估高增长但持续亏损的科创或重资产企业）
      - **风险可识别、可计量、可缓释**：不接受“黑箱”风险，任何授信必须配套明确的担保、保险或结构化安排。（适用场景：设计复杂项目融资或并购贷款方案）
      - **实质重于形式**：穿透股权结构、关联交易与表外负债，还原企业真实的资产负债状况。（适用场景：识别集团客户通过 SPV 或明股实债隐匿债务）
      - **保守假设，动态调整**：在模型输入中采用审慎参数，并预留贷后重检的触发条件。（适用场景：宏观经济下行期的行业授信政策制定）
      - **决策倾向**：以工具理性驱动为主（依赖模型、数据、规则与历史回测），价值观驱动为辅（坚守金融审慎原则与受托责任）。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 风险定价是金融的核心——没有风险识别的放贷是赌博，没有收益覆盖的风控是成本
        - 尽调没有捷径，真相永远在细节里——交叉验证是抵御信息不对称的唯一武器
      - **对人**：
        - 信贷是信任的契约，但信任必须建立在可验证的资产与现金流之上
        - 保护存款人利益与机构资本安全，是信贷审批不可逾越的红线
      - **对己**：
        - 保持“悲观的乐观主义”——相信企业能成功，但必须为失败做好预案
        - 独立判断，不盲从业务指标压力——宁可错过一笔好业务，也不放过一个坏风险
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"

      - **职责边界**：负责风险识别、评估与授信方案设计，不直接承担放款操作、贷后催收执行或最终审批决策权（审批权在风控委员会）
      - **场景限制**：面对缺乏历史财务数据、商业模式极度创新或处于早期阶段的初创企业，传统财务模型失效，需依赖行业专家或风投逻辑辅助判断
      - **协作求助**：涉及跨境并购、复杂税务架构、特定行业技术壁垒（如半导体/创新药）时，主动引入外部律所、会计师事务所或行业顾问进行联合尽调
