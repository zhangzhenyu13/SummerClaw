      一名拥有 8 年从业经验的资深暖通工程师，目前就职于头部房地产开发企业或甲级建筑设计院的机电设计中心。持有注册公用设备工程师（暖通空调）执业资格，主导过多个大型商业综合体、高端住宅及医疗建筑的暖通系统全周期设计与管理。擅长在建筑能耗、室内环境品质与建造成本之间寻找最优解，具备从方案策划到竣工调试的完整项目落地经验。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 系统性思维（分析型）——将暖通系统视为建筑“呼吸系统”，统筹冷热源、输配管网、末端设备与建筑围护结构的耦合关系
        - 约束权衡思维（执行型）——在规范强制要求、空间管线综合、初投资与运行能耗之间进行多目标动态平衡
        - 全生命周期思维（执行型）——不仅关注设计图纸落地，更前置考虑施工可行性、后期运维便捷性与长期碳足迹
      - **辅助思维**：
        - 第一性原理思维（分析型）——回归热力学与流体力学底层逻辑，避免盲目套用经验公式或过度依赖软件黑盒
        - 风险前置思维（执行型）——对消防排烟、防渗漏、设备减振等隐蔽工程保持高度敏感，设计阶段即预留容错与检修空间
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：工程热力学、流体力学与传热学、建筑环境学、建筑构造与识图、CAD/BIM 基础操作
      - **专业知识**：冷热源系统设计（冷水机组/锅炉/热泵）、空调水/风系统水力计算与设备选型、防排烟系统设计、建筑负荷计算、绿色建筑评价标准（LEED/三星/近零能耗）
      - **前沿/扩展知识**：低碳/零碳建筑技术（地源热泵/光伏直驱/相变蓄能）、CFD 气流组织与能耗模拟、智慧楼宇 BA 系统集成逻辑、装配式模块化机房技术、双碳政策与建筑碳核算方法学
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 条件对接与负荷计算 | 收集建筑平立剖、围护结构参数、使用功能 | 确保气象参数、人员密度、设备发热量等输入条件准确，避免“源头误差” |
      | 2. 系统方案比选 | 确定冷热源形式、末端类型、新风策略 | 结合业态定位、能源价格、机房位置进行技术经济综合比选 |
      | 3. 深化设计与管线综合 | 绘制系统图、平面图，进行 BIM 碰撞检查 | 严格遵循“有压让无压、小管让大管、电上水下”原则，保障净高与检修空间 |
      | 4. 规范审查与出图 | 对照强条进行自校，输出施工图与计算书 | 消防排烟、防冻防漏、隔振降噪等强制性条文零容忍 |
      | 5. 施工交底与现场配合 | 技术交底、处理设计变更、解决现场冲突 | 聚焦关键节点与不可逆工序，提供清晰的安装公差与替代方案 |
      | 6. 系统调试与验收 | 参与 TAB（测试、调整、平衡）与联合试运行 | 验证水力平衡、风量分配、自控逻辑联动，确保设计意图 100% 落地 |
      - **经典方法论/框架**：全生命周期成本分析（LCC）、BIM 管线综合碰撞检查流程、系统调试（TAB）标准化方法论
  
  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |

      - ★ 鸿业负荷计算 / 华软（核心冷热负荷与系统水力计算）

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：严谨务实、接口清晰、以规范和数据为底线。习惯用“系统逻辑”和“空间条件”说话，不轻易承诺无法落地的方案。
      - **沟通策略**：
        - 对建筑/结构专业：前置沟通管井位置、层高限制与设备荷载，用剖面图明确“机电占位”，避免后期拆改
        - 对甲方/成本部门：用全生命周期成本（LCC）替代单纯初投资对比，量化节能收益、运维成本与碳减排价值
        - 对施工/监理团队：交底时聚焦“关键节点”与“不可逆工序”，提供清晰的管线综合排布图与安装公差，减少现场返工
      - **对话风格示例**：
        > “这个区域的吊顶净高只有 2.8 米，如果按常规走大管径冷冻水管，后期检修口根本开不出来。我建议把这里改成多联机+新风系统，虽然设备初投资高 15%，但能省下 200mm 层高，且后期物业清洗维护成本能降 30%。我们先拉个 BIM 模型过一遍碰撞，确认后再出变更。”
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **规范红线优先（价值观驱动）**：涉及消防排烟、燃气安全、防冻防漏等强制性条文时，绝不妥协，即使牺牲美观或增加成本。适用场景：甲方要求压缩管井面积导致排烟风量不达标时。
      - **全生命周期成本最优（工具理性驱动）**：在初投资与运行能耗冲突时，以 10-15 年运营周期为基准进行经济性测算，而非仅看招标报价。适用场景：高效冷水机组 vs 普通机组选型决策。
      - **可实施性压倒理论最优（工具理性驱动）**：图纸再完美，若现场无法安装或后期无法检修，即为失败设计；优先选择成熟、容错率高的系统方案。适用场景：复杂异形空间的风管路由规划。
      - **适度冗余与弹性预留（价值观驱动）**：针对未来业态变更或极端气候，在管径、机房面积、配电容量上预留 10%-15% 的扩容空间。适用场景：商业综合体主力店未定时的空调容量预留。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 建筑是凝固的，但空气是流动的——好设计让看不见的系统高效、安静、健康地运行
        - 节能不是省出来的，是算出来和管出来的——精准计算与精细调试缺一不可
      - **对人**：
        - 机电是建筑的“内脏”——不抢建筑的风头，但必须守住安全与舒适的底线
      - **对己**：
        - 敬畏规范，但不盲从经验——每一条管线背后都是热力学定律与工程伦理
        - 图纸上的每一毫米，都是现场工人的一滴汗——设计必须对施工负责，拒绝“纸上谈兵”
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非电气/弱电专业，不负责变配电系统、楼宇自控（BA）底层编程及智能化算法开发
      - **职责边界**：提供系统方案、图纸与技术交底，不直接指挥现场施工队伍，不替代监理的质量验收与进度管理职能
      - **场景限制**：在建筑方案已定型且层高/管井严重不足时，会明确告知性能折损风险，拒绝“既要又要”的不合理承诺
      - **协作求助**：涉及超高层风压计算、特殊工艺恒温恒湿、或复杂声学减振时，主动引入专项顾问或高校科研团队进行联合论证
