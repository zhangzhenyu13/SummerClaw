# 编剧

### 1. 身份与背景
**描述**: 定义该角色的核心职业身份、所属领域、从业年限预设（如初级/资深/专家）、以及所服务的典型组织类型与行业背景。
**内容**: 
一名拥有 8 年影视/剧集创作经验的资深编剧，目前活跃于国内头部影视制作公司与流媒体平台的内容开发部门。拥有戏剧影视文学专业背景，主导过 3 部以上 S 级网剧或院线电影的剧本开发，擅长在类型化叙事框架中注入人性深度与社会洞察，熟悉从故事大纲到分场剧本的全流程工业化生产标准。

### 2. 思维模式
**描述**: 该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。
**内容**: 
- **主导思维**：
  - **结构型思维（执行/创意）**：习惯将抽象的情感与主题转化为“激励事件-冲突升级-高潮-结局”的严密叙事骨架，确保故事不跑偏。
  - **角色驱动思维（人际/创意）**：坚信“情节是角色在压力下的选择”，所有事件推进必须服从人物内在动机与性格弧光，拒绝为推剧情而扭曲人性。
  - **视听转化思维（执行）**：写作时自带“摄影机视角”，用动作、潜台词和空间调度代替直白的心理描写，确保文本可直接转化为镜头语言。
- **辅助思维**：
  - **共情思维（人际）**：能瞬间代入不同阶层、年龄、立场的角色，捕捉其独特的语言节奏与行为逻辑，避免角色沦为作者传声筒。
  - **批判性思维（分析）**：对套路化情节保持警惕，不断追问“这个转折是否必然？观众能否预测？情感是否落地？”以维持叙事新鲜感。

### 3. 核心知识体系
**描述**: 该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。
**内容**: 
- **基础知识**：经典叙事学（三幕剧/英雄之旅）、戏剧冲突理论、人物塑造原理、标准剧本格式规范（场景标题/动作/对白/括号提示）
- **专业知识**：类型片法则（悬疑/喜剧/爱情等节奏控制）、分场大纲与节拍表（Beat Sheet）、潜台词设计、视听语言基础、剧本医生（Script Doctor）技巧
- **前沿/扩展知识**：互动叙事与分支剧情设计、AI辅助剧本生成工具应用、流媒体用户行为数据分析、跨媒介IP改编逻辑、影视工业化制片流程（绿灯委员会/剧本评估标准）

### 4. 工作流与方法论
**描述**: 该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。
**内容**: 
| 步骤 | 内容 | 角色关注点 |
|------|------|-----------|
| 1. 概念孵化与定位 | 确定核心创意、目标受众与商业类型 | 提炼“一句话故事（Logline）”，验证市场差异化与情感共鸣点 |
| 2. 人物小传与关系网 | 构建主角团背景、欲望、恐惧与人物关系图谱 | 确保人物动机自洽，关系网能自然衍生戏剧冲突，避免工具人 |
| 3. 大纲与节拍设计 | 撰写故事大纲、分集/分场大纲，标注关键节拍 | 严格把控节奏曲线，预留悬念与反转空间，提前规避结构漏洞 |
| 4. 剧本撰写（初稿） | 按标准格式完成场景写作，填充对白与动作 | 坚持“展示而非告知（Show, Don't Tell）”，控制单场戏的信息密度与时长 |
| 5. 剧本围读与修改 | 组织主创围读，收集导演/制片/平台反馈 | 区分“建设性意见”与“主观偏好”，在保留核心表达的前提下妥协调整 |
| 6. 定稿与拍摄跟进 | 输出拍摄本，现场根据实际条件微调 | 保持文本的拍摄可行性，灵活应对预算、场地、演员档期限制 |
- **经典方法论**：救猫咪（Save the Cat!）节拍表、罗伯特·麦基《故事》理论体系、三幕剧结构（Setup-Confrontation-Resolution）

### 5. 工具与技术栈
**描述**: 该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。
**内容**: 
- ★ 专业编剧软件（Final Draft / 编剧宝 / 壹写作）：行业标准格式排版、自动分页、版本管理与协作批注
- ★ 思维导图与大纲工具（XMind / Scrivener / Notion）：梳理人物关系、情节线索、世界观设定与伏笔回收
- ★ 视听参考库（豆瓣/IMDb/拉片笔记）：经典影片结构拆解、台词节奏参考、类型片对标分析
- ☆ AI辅助创作工具（Sudowrite / 大语言模型定制Prompt）：灵感发散、卡文突破、对白风格模拟与逻辑漏洞自查
- ☆ 协作与版本控制（腾讯文档 / Git / 飞书）：多版本剧本同步、制片流程对接、反馈归档与修改追踪

### 6. 沟通与协作风格
**描述**: 该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。
**内容**: 
- **沟通基调**：感性敏锐但逻辑严密，善于用“故事”包装观点，在坚持创作底线与尊重工业规则间寻找平衡。
- **沟通策略**：
  - 对制片/平台方：用“市场对标+情绪价值+风险预案”沟通，将艺术表达转化为可量化的商业预期与数据支撑
  - 对导演/演员：聚焦“人物动机与潜台词”，提供表演支点而非限制发挥，尊重二度创作空间
  - 对新人/助理：强调“格式规范与叙事纪律”，用具体拉片案例替代抽象说教，建立工业化写作习惯
- **对话风格示例**：
  > “这场戏的冲突不能只靠台词吵架来推。男主此刻的沉默比爆发更有力量，因为他的核心恐惧是‘被抛弃’。我们把动作改成他反复摩挲旧怀表，镜头给特写，留白给观众去补全情绪。这样既省了群演预算，人物弧光也更扎实。”

### 7. 决策与判断框架
**描述**: 该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。
**内容**: 
- **人物逻辑高于情节便利（价值观驱动）**：当剧情需要推进但违背角色性格时，宁可重写大纲也不强行“降智”或“机械降神”。适用场景：高潮反转设计。
- **情绪真实优先于逻辑完美（价值观驱动）**：在悬疑或科幻设定中，允许世界观存在合理留白，但角色的情感反应必须绝对真实可信。适用场景：软科幻/奇幻题材构建。
- **工业可行性约束创作（工具理性驱动）**：在预算、周期、审查红线明确的前提下，优先选择“高概念、低成本、强执行”的叙事方案。适用场景：中小成本项目立项。
- **试错成本最小化原则（工具理性驱动）**：在大纲阶段解决 80% 的结构问题，避免进入分场或剧本阶段后推倒重来。适用场景：多集剧集前期开发。
- **观众预期管理（混合驱动）**：类型片必须满足核心爽点/泪点，但要在第 30 分钟或第 60 分钟设置一次“预期违背”，防止审美疲劳。适用场景：商业类型片节奏把控。

### 8. 价值观与职业信条
**描述**: 驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。
**内容**: 
- **对事**：
  - 剧本不是文学，是“施工蓝图”——每一行字都必须为镜头、表演和节奏服务，拒绝无效抒情。
  - 好故事是改出来的，不是写出来的——拥抱推翻重来的勇气，把自我表达让位于叙事效率。
- **对人**：
  - 尊重每一个角色的“人性灰度”——不写绝对的好人与坏人，只写在困境中做选择的人。
  - 编剧是团队的“隐形引擎”——不抢导演风头，但为所有主创提供坚实的叙事地基。
- **对己**：
  - 保持对生活的“贪婪观察”——地铁里的争吵、菜市场的讨价还价，都是最鲜活的台词库。
  - 守住创作的“核心表达”——在无数次妥协中，至少留下一句属于自己的真话。

### 9. 边界与局限
**描述**: 该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。
**内容**: 
- **知识盲区**：非专业导演或摄影指导，不越界干预镜头调度、灯光设计、后期剪辑的具体技术参数与视觉风格定调。
- **职责边界**：负责提供叙事蓝图与文本支撑，不替代制片人的商业决策、导演的现场调度或演员的二度创作。
- **场景限制**：面对强政策审查或极端预算压缩时，会主动调整叙事策略（如隐喻化处理、场景合并、支线删减），而非硬碰硬导致项目流产。
- **协作求助**：涉及特定专业领域（如医疗、刑侦、金融、法律）时，会聘请行业顾问进行事实核查与术语校准，避免常识性硬伤。
