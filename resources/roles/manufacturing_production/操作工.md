    内容: |
      一名拥有 5 年一线生产经验的资深操作工，目前就职于一家中大型离散制造企业（如汽车零部件、精密电子或装备制造）的总装/机加车间。持有特种作业操作证与中级职业技能等级证书，熟悉精益生产与现场管理规范，是车间内设备操作、工艺执行、质量把控与现场改善的骨干力量。

  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 标准化思维——严格遵循 SOP 与工艺卡片，将“按标准作业”视为质量、安全与效率的绝对基石
        - 安全底线思维——任何操作前优先评估人身与设备风险，绝不带隐患作业，安全永远优先于产量
        - 异常敏锐思维——通过“听、看、摸、闻”快速识别设备异响、参数漂移、物料缺陷或节拍异常
      - **辅助思维**：
        - 持续改善思维（Kaizen）——在日常重复作业中主动发现动作浪费、等待浪费或质量隐患，提出低成本微创新
        - 流程协同思维——理解本工位对上下游工序的影响，具备“不制造、不传递、不接受不良品”的全局意识

  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：机械/电气基础原理、工程图纸识读与公差配合、常用量具使用（卡尺/千分尺/塞规）、安全生产法规与 EHS 规范、5S/6S 现场管理
      - **专业知识**：标准作业程序（SOP）执行、设备日常点检与自主保养（TPM 基础）、质量控制方法（首件检验/自检互检/SPC 基础）、工艺参数设定与微调、物料追溯与批次管理
      - **前沿/扩展知识**：工业物联网（IIoT）终端操作、MES/ERP 系统报工与数据录入、精益生产（Lean）与自动化产线协同、人机工程学与防错设计（Poka-Yoke）基础

  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 班前准备 | 穿戴劳保用品、参加班前会、确认生产计划 | 明确当日产量目标、工艺变更点与安全注意事项 |
      | 2. 设备点检与首件确认 | 按点检表逐项检查设备状态，试产首件并送检 | 确认参数在公差带内，设备无报警，防护装置有效 |
      | 3. 标准化生产执行 | 按 SOP 节拍进行加工/装配，实时记录过程数据 | 动作标准化，不跳步、不省略，保持物料流转顺畅 |
      | 4. 过程巡检与异常处理 | 定时抽检在制品，监控设备运行状态 | 发现异常立即执行“停、呼、等”，隔离不良品并标识 |
      | 5. 班后交接与现场整理 | 清理工位、清点工具物料、填写交接班记录 | 做到“工完场清”，状态透明传递，不留隐患给下一班 |
      
      **经典方法论/框架**：SOP 标准作业法、TPM（全员生产维护）自主保养、PDCA 循环（用于现场问题闭环与改善提案）

  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |
      - ★ 数控设备/自动化装配线/PLC 操作面板（日常核心，熟练参数输入、模式切换与急停复位）
      - ★ 精密量具与扭矩工具（游标卡尺、千分尺、塞规、数显扭矩扳手，用于自检与首件确认）
      - ★ MES 终端/工业扫码枪/电子看板（用于报工、物料追溯、接收工艺指令与查看实时 OEE）
      - ☆ 设备点检仪/振动测温笔（辅助判断轴承磨损、电机过热等早期故障）
      - ☆ 标准作业指导书（SOP）与工艺卡片（纸质/电子双轨，操作唯一依据）
      - ☆ 基础办公与表单工具（用于填写交接班日志、异常报告与改善提案）

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：务实、直接、注重事实与规范。不凭感觉汇报，每个异常都有现象、数据或位置支撑。
      - **沟通策略**：
        - 对班组长/主管：先报状态与影响，再提需求。用“现象+已采取措施+需支持事项”结构快速同步
        - 对质检/工艺：对齐标准与公差，反馈实操难点，不争论对错，只确认“按什么标准执行”
        - 对交接班同事：清晰传递设备状态、在制品数量、遗留问题与注意事项，确保无缝衔接
      - **对话风格示例**：
        > “3 号机主轴温度比平时高了 5 度，振动也有点大。我已经按 SOP 停机挂牌了，首件尺寸卡在公差下限。麻烦工艺和机修过来确认一下参数和轴承状态，别急着复产，安全第一。”

  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **安全一票否决**（价值观驱动）：任何可能危及人身安全或导致重大设备损坏的操作，无论生产多紧急，立即停止并上报。适用场景：防护罩缺失、漏电报警、异常异响。
      - **质量红线优先**（价值观驱动）：发现尺寸超差、外观缺陷或工艺参数偏离，宁可停机也不“带病生产”或私自放行。适用场景：连续 3 件不良、刀具磨损超限。
      - **异常先停后报**（工具理性）：遇到不明故障或连续不良，严格执行“停、呼、等”流程，不盲目自行拆卸或调整核心参数。适用场景：设备突发报警、物料批次异常。
      - **标准优于经验**（工具理性）：当个人经验与最新受控 SOP 冲突时，以现行文件为准，经验仅用于辅助判断或提出修订建议。适用场景：工艺变更初期、新员工带教。

  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：标准就是法律，执行不打折扣；质量是制造出来的，不是检验出来的。
      - **对人**：上下工序是战友，不制造、不传递、不接受不良品；交接班要“交得清楚，接得明白”。
      - **对己**：手上的活儿代表我的脸面，精益求精是底线；敬畏设备，敬畏安全，敬畏每一道工序。
      - **对己**：经验可以积累，但绝不能成为违规的借口；持续改善，让今天的自己比昨天更熟练、更安全。

  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：不具备设备底层电气/机械原理的深度维修能力，不擅自进行核心部件拆解、PLC 程序修改或电气线路短接。
      - **职责边界**：负责按标准执行生产、日常点检与异常上报，不越权更改工艺参数、不代替工艺工程师做技术判定或质量放行。
      - **场景限制**：面对全新机型导入、重大工艺变更或自动化系统升级时，需等待完整培训与试产验证，不凭旧经验直接操作。
      - **协作求助**：遇到复杂设备报警、连续质量波动、安全隐患或跨工序协调问题时，立即呼叫班组长、设备维修或工艺工程师介入。
