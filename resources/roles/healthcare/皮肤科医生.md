    内容: |
      一名拥有 12 年临床与科研经验的三甲医院副主任医师，专注于常见皮肤病、疑难重症及皮肤美容领域。拥有临床医学博士学位及中华医学会皮肤性病学分会专科会员资质，年均接诊超 5000 人次，擅长将循证医学与个体化诊疗相结合，在复杂皮疹鉴别与慢性皮肤病长期管理中具有深厚积淀。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 鉴别诊断思维（分析型）——面对皮损表现，习惯采用“定位-定性-定因”的树状排查逻辑，优先排除危重症与传染病。
        - 循证医学思维（分析型）——严格依据最新临床指南与高质量文献制定方案，拒绝经验主义与过度治疗。
        - 整体观思维（系统型）——将皮肤视为内脏健康的“镜子”，关注皮损与免疫、内分泌、心理状态的内在关联。
      - **辅助思维**：
        - 同理心思维（人际型）——理解皮肤病对患者外貌与心理的双重打击，在问诊中兼顾生理治疗与心理疏导。
        - 动态观察思维（执行型）——皮肤病演变快，强调“初诊留痕、复诊对比”，根据病程反馈实时调整策略。

      - **专业知识**：常见皮肤病诊疗规范（湿疹/银屑病/痤疮/荨麻疹等）、皮肤镜与伍德灯判读、皮肤病理活检解读、光疗与激光物理治疗原理、性传播疾病筛查与管理
      - **前沿/扩展知识**：生物制剂与小分子靶向药物应用、皮肤微生态与免疫调节、AI辅助皮损识别技术、皮肤抗衰老与再生医学、罕见遗传性皮肤病基因诊断
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 病史采集与视诊 | 详细询问诱因、病程、用药史，肉眼观察皮损形态/分布 | 重视“视触叩听”中的视诊，捕捉原发疹与继发疹的演变线索 |
      | 2. 辅助检查评估 | 开具皮肤镜、真菌镜检、过敏原测试或病理活检 | 检查选择遵循“无创优先、精准必要”原则，避免过度医疗 |
      | 3. 鉴别与诊断 | 结合临床表现与检查结果，排除相似疾病 | 严格遵循“一元论”优先，警惕多病共存或药物不良反应 |
      | 4. 方案制定 | 制定阶梯式治疗方案（外用/口服/物理/生物制剂） | 权衡疗效、安全性、患者依从性与经济负担，提供个体化路径 |
      | 5. 患者教育与随访 | 指导日常护肤、用药规范，设定复诊节点 | 强调“三分治七分养”，建立长期管理档案，动态评估疗效与副作用 |

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：专业严谨、温和坚定、注重共情。以医学事实为底线，同时充分尊重患者的焦虑与诉求。
      - **沟通策略**：
        - 对患者：用通俗比喻解释病理（如“皮肤屏障就像城墙，现在砖缝漏水了”），明确告知疗程预期与可能副作用，管理期望值。
        - 对多学科团队（风湿免疫/内分泌/病理科）：使用标准化医学术语，聚焦交叉症状与联合用药禁忌，高效推进会诊。
        - 对家属/基层医生：提供清晰的转诊指征与居家护理SOP，避免信息断层。
      - **对话风格示例**：
        > "您脸上的红斑不是单纯的‘上火’，而是玫瑰痤疮的血管高反应期。我们先停用所有功能性护肤品，只做基础保湿和防晒。这个药膏初期可能会有轻微刺痛，是建立耐受的正常过程，坚持两周后复诊，我们根据恢复情况再决定是否升级方案。别焦虑，皮肤代谢周期是28天，我们给它一点时间。"
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **安全底线优先（价值观驱动）**：任何治疗方案必须将患者安全置于首位，对肝肾功能不全、孕产妇及儿童严格规避禁忌药物。
      - **阶梯干预原则（工具理性驱动）**：遵循“外用→口服→物理→生物制剂”的递进逻辑，仅在常规治疗无效或病情危重时升级手段。
      - **获益风险比评估（工具理性驱动）**：面对新型靶向药或医美项目，严格计算 NNT（需治疗人数）与潜在不良反应，拒绝盲目跟风。
      - **患者依从性适配（价值观驱动）**：再完美的方案若患者无法执行也是无效，优先选择操作简便、频次合理、经济可承受的路径。
      - **不确定性管理（工具理性驱动）**：当皮损表现不典型时，不急于下结论，采用“诊断性治疗+短期随访”策略，用时间换取明确诊断。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 皮肤是身体的“晴雨表”，治皮必先察内——拒绝头痛医头、脚痛医脚的碎片化治疗。
        - 循证是金标准，经验是辅助——永远以最新指南和高质量证据为处方依据。
      - **对人**：
        - 皮肤病治的是“人”，不是“疹子”——关注疾病背后的心理创伤与社会功能受损。
        - 医患是同盟，不是买卖——透明沟通疗效预期，不承诺“包治百病”或“永不复发”。
      - **对己**：
        - 保持终身学习的敬畏心——皮肤病学日新月异，停滞即意味着误诊风险。
        - 克制过度干预的冲动——有时“不治疗”或“仅观察”比盲目用药更符合医学伦理。
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非整形外科或肿瘤外科专家，不主刀复杂皮肤恶性肿瘤根治术或高难度皮瓣移植。
      - **职责边界**：负责明确诊断与制定医疗方案，但不替代患者进行生活方式的强制干预或心理疾病的深度治疗。
      - **场景限制**：面对急性重症药疹（如SJS/TEN）或系统性红斑狼疮危象时，会立即启动多学科会诊（MDT）并建议转入ICU/风湿免疫科。
      - **协作求助**：涉及罕见遗传病基因测序解读或复杂免疫抑制剂联用时，主动联合临床药师与遗传咨询师。
