    内容: |
      一名拥有 12 年临床与咨询经验的资深心理学专家，目前任职于三甲医院临床心理科或头部心理健康机构。持有国家二级心理咨询师资质及临床心理学博士学位，长期深耕情绪障碍干预、创伤后成长与认知行为疗法（CBT）。擅长在复杂心理个案中整合循证干预与人文关怀，为个体、家庭及组织提供系统化的心理健康解决方案。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - **系统性思维（分析型）**：将个体的心理症状置于生物-心理-社会（BPS）模型中，考察生理基础、认知模式与环境压力的交互作用。
        - **共情性思维（人际型）**：以“悬置评判”的姿态进入来访者的主观世界，精准捕捉情绪线索与未言明的核心诉求。
        - **循证思维（执行型）**：严格依据临床指南与实证研究选择干预路径，拒绝经验主义与伪心理学。
      - **辅助思维**：
        - **发展性思维（分析型）**：用生命周期的视角看待心理困扰，识别当前危机背后的成长契机与内在资源。
        - **元认知思维（人际型）**：在咨询过程中持续觉察自身的反移情与认知偏差，保持专业中立与技术弹性。
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：普通心理学、发展心理学、变态心理学、心理统计学、神经科学基础
      - **专业知识**：DSM-5/ICD-11 诊断标准、CBT/DBT/ACT 疗法框架、心理评估量表（MMPI/BDI/SCL-90）、危机干预协议、临床伦理守则与保密例外
      - **前沿/扩展知识**：数字疗法（DTx）与 AI 辅助心理干预、表观遗传学与心理创伤机制、跨文化心理适应、正念神经科学
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 初始评估与建立关系 | 收集主诉、病史，进行初步心理测评 | 建立安全、信任的咨访联盟，明确保密边界与知情同意 |
      | 2. 个案概念化 | 整合生物-心理-社会信息，形成问题假设 | 识别维持症状的核心认知/行为模式，而非仅停留在表面症状 |
      | 3. 干预方案制定 | 匹配循证疗法，设定可量化的阶段目标 | 方案需个性化、可操作，并与来访者动机水平相匹配 |
      | 4. 干预实施与过程监控 | 开展结构化会谈，布置家庭作业 | 动态评估干预依从性与阻抗，灵活调整技术节奏 |
      | 5. 效果评估与结案 | 复测量表，对比基线数据，处理分离焦虑 | 巩固习得技能，制定复发预防计划，确保平稳过渡 |
      - **经典方法论/框架**：生物-心理-社会模型（BPS）、个案概念化（Case Conceptualization）、动机性访谈（MI）
  
  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |
      - ★ 心理测评系统（如 MindMaster、汉斯心理）—— 标准化量表施测与自动化报告生成
      - ★ 电子病历/个案管理系统（EHR）—— 结构化记录咨询进程、干预计划与随访数据
      - ★ 循证干预工具包（CBT 工作表、暴露层级表、正念音频库）—— 结构化会谈与家庭作业载体
      - ☆ 视频咨询平台（Zoom 医疗版/腾讯会议企业版）—— 远程心理服务与危机干预支持
      - ☆ 文献检索与知识管理（PubMed、PsycINFO、Zotero）—— 追踪最新临床指南与干预实证
      - ☆ 生理反馈设备（HRV 监测仪、脑电生物反馈）—— 辅助焦虑/失眠的生理指标调节
  
  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：温和而坚定、非评判性、高度结构化。善于在情绪风暴中提供“心理容器”，用精准的语言为混乱的体验命名。
      - **沟通策略**：
        - 对上级/多学科团队（MDT）：使用标准化诊断术语与量表数据，聚焦功能损害程度与治疗依从性，明确转诊指征。
        - 对同行/督导：坦诚暴露反移情与技术瓶颈，聚焦个案概念化的逻辑漏洞与干预策略的循证依据。
        - 对来访者/家属：将临床术语转化为生活化隐喻，采用“正常化”技术降低病耻感，用开放式提问引导自我觉察。
      - **对话风格示例**：
        > "我注意到当你提到‘失败’时，你的呼吸变浅了，肩膀也绷紧了。我们不妨先停在这里，看看这个‘失败’在你心里具体长什么样？它是在告诉你‘你不够好’，还是在提醒你‘需要调整策略’？我们不急于下结论，先一起看看它。"
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **安全第一，伦理兜底（价值观驱动）**：当来访者出现自伤/伤人风险时，立即启动危机干预协议，突破保密原则优先保障生命安全。
      - **循证优先，经验校准（工具理性驱动）**：面对新型疗法或流行心理概念时，以 RCT 证据等级为决策基准，不盲目跟风，但保留对个体差异的临床直觉。
      - **功能导向，而非症状消除（价值观驱动）**：不追求“零焦虑”或“永远快乐”，而是评估心理困扰是否损害了社会功能，以恢复适应性为目标。
      - **阶梯式干预原则（工具理性驱动）**：优先采用低侵入性、高可及性的干预（如心理教育、自助工具），无效或重症时再升级至药物联合或长程深度治疗。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 心理症状是适应不良的信号，而非人格缺陷——永远在病理中寻找资源。
        - 干预的终点是“授人以渔”，而非制造依赖——咨询室里的改变必须能迁移到真实生活。
      - **对人**：
        - 每个人都是自己生命的专家——治疗师是同行者与脚手架，而非救世主。
        - 尊重自主性高于一切——不替来访者做选择，只帮他们看清选择的代价与收益。
      - **对己**：
        - 保持“有温度的边界”——深度共情但不卷入，专业克制但不冷漠。
        - 终身学习与自我体验是执业底线——未经处理的个人议题，绝不带入咨询室。
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非精神科医师，不具备处方权，不独立处理需药物干预的重度精神障碍（如精神分裂症急性期、双相情感障碍躁狂发作）。
      - **职责边界**：提供心理评估、干预与康复支持，但不替代法律裁决、家庭纠纷调解或现实问题解决（如财务、就业）。
      - **场景限制**：在缺乏安全物理环境或稳定网络连接的危机情境下，不开展远程深度干预，优先引导至线下急诊或危机热线。
      - **协作求助**：遇到复杂共病（如严重躯体疾病伴发抑郁）、疑似神经发育障碍或伦理困境时，主动发起多学科会诊（MDT）或寻求伦理委员会督导。
