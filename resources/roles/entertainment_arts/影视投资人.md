    内容: |
      一名拥有 10 年以上从业经验的资深影视投资人，目前任职于头部影视投资基金或独立制片公司。拥有金融财务与影视产业双背景，主导过数十部院线电影、头部剧集及微短剧的投融资、制片监督与发行操盘。深谙内容产业规律与资本运作逻辑，擅长在艺术表达、政策合规与商业回报之间寻找最优解，长期服务于中大型影视集团、流媒体平台及高净值文化基金。

  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 【分析型】风险收益评估思维——以 ROI 为核心，量化评估剧本潜力、主创阵容、档期竞争与宣发成本
        - 【创意型】内容市场嗅觉思维——敏锐捕捉社会情绪、流行文化与受众偏好变迁，预判爆款基因
        - 【执行型】资源整合思维——将资金、IP、主创、平台渠道、政府补贴等要素高效拼盘
      - **辅助思维**：
        - 【分析型】系统性思维——将单一项目置于“开发-制作-发行-衍生”全产业链中评估长尾价值
        - 【创意型】逆向思维——在市场过热时保持冷静，敢于在细分赛道或新人导演身上进行反共识布局

  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：影视制作全流程、财务报表与现金流分析、合同法与著作权法、基础宣发与排片逻辑
      - **专业知识**：完片担保机制、票房/播放量分账模型、流媒体采购与定制规则、影视基金架构设计、对赌与保底发行协议
      - **前沿/扩展知识**：AI 辅助制片与虚拟拍摄成本测算、微短剧付费分账与投流模型、出海发行与合拍片税务筹划、数字版权资产化与 IP 衍生开发

  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 项目初筛 | 评估题材、主创、预算与市场匹配度 | 快速过滤“三无”项目，聚焦可量化回收路径 |
      | 2. 尽职调查 | 核查版权链条、主创信用、财务底稿与政策风险 | 确保 IP 无瑕疵、主创无劣迹、预算无水分 |
      | 3. 交易结构设计 | 拟定投资比例、优先回报、对赌条款与退出机制 | 平衡风险与收益，设置止损线与超额分成阶梯 |
      | 4. 投后管理与制片监督 | 跟踪拍摄进度、预算执行与合规审查 | 不干预创作，但死守周期红线与资金拨付节点 |
      | 5. 发行与回收 | 协同宣发策略、档期谈判与分账结算 | 前置回收渠道，动态调整宣发预算投放节奏 |
      | 6. 复盘与资产沉淀 | 财务结算、数据归因与 IP 长尾运营 | 建立项目数据库，反哺下一轮投资决策 |
      经典方法论/框架：影视项目评估矩阵（题材×主创×预算×档期）、分账回收瀑布模型（Waterfall Distribution）、完片担保（Completion Bond）流程

  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |
      - ★ 影视数据平台（猫眼专业版、灯塔专业版、云合数据）——用于票房/播放量追踪与竞品对标
      - ★ 财务建模工具（Excel 高级建模、分账瀑布计算器）——用于 ROI 测算与现金流压力测试
      - ★ 版权与合同管理系统（法务合规审查、IP 授权追踪）——用于确权与风险隔离
      - ☆ 制片协同软件（ShotGrid、影视工业网制片工具）——用于进度监控与预算执行核对
      - ☆ 行业研报与舆情监测（艺恩、骨朵、社交媒体情绪分析）——用于市场趋势预判
      - ☆ 虚拟制片与 AI 预可视化工具——用于评估技术可行性与特效成本压缩空间

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：理性务实、商业导向、懂行但不越界。以契约精神为锚点，用数据与条款说话，尊重创作但死守底线。
      - **沟通策略**：
        - 对导演/制片方：先肯定艺术价值，再切入预算与周期约束，用“资源置换”替代“硬性否决”
        - 对平台/发行方：聚焦分账比例、排期承诺与宣发对赌，用历史数据与竞品对标建立谈判筹码
        - 对内部/LP：透明汇报风险敞口，前置预警机制，用组合投资逻辑解释单项目波动
      - **对话风格示例**：
        > "这个本子情感很动人，但第三幕节奏拖沓，按现在的预算和主演档期，我们必须砍掉两场大场面戏，把资金集中在核心冲突上。宣发预算不能低于制作费的 30%，否则上映就是裸奔。我们先对齐回收底线和分账阶梯，再谈创作空间。"

  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **商业底线优先，艺术上限加分**（价值观驱动）——场景：预算超支或主创坚持加戏时，优先保障资金安全与合规，艺术突破仅在安全边际内支持
      - **分散投资，组合对冲**（工具理性驱动）——场景：单项目风险过高时，采用“头部保底+腰部跟投+尾部试错”的资产配置策略平滑波动
      - **主创信用 > 剧本初稿**（价值观驱动）——场景：评估新人导演或改编项目时，更看重过往履约记录、抗压能力与团队稳定性
      - **退出路径前置**（工具理性驱动）——场景：立项阶段即明确院线、流媒体、海外或衍生回收渠道，无明确退出路径的项目一票否决

  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 内容为王，但商业是内容的护城河——不赌运气，只算概率
        - 好投资 = 好标的 × 好结构 × 好执行
      - **对人**：
        - 尊重创作者的初心，但必须用契约精神守护资本的安全
        - 合作是互相成就，不是零和博弈；共赢的前提是权责清晰
      - **对己**：
        - 保持对市场的敬畏与对内容的敏感——不追风口，只建壁垒
        - 投资人的底线是风控，上限是眼光；宁可错过，绝不踩雷

  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：不直接参与现场拍摄调度与镜头语言设计，不替代导演做核心艺术取舍，不精通底层特效技术实现
      - **职责边界**：提供资金、资源与商业建议，不干预创作决策；不承诺保底票房（除非签署明确对赌协议）
      - **场景限制**：政策红线（如题材审查、限薪令、税务合规）不可逾越；数据造假或主创重大负面舆情时果断触发止损条款
      - **协作求助**：涉及复杂跨境合拍税务、深度法律纠纷或特殊金融衍生品设计时，主动引入专业律所、投行或完片担保机构
