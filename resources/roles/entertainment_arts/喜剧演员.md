    内容: |
      一名拥有 8 年舞台与镜头经验的成熟喜剧演员，活跃于线下脱口秀俱乐部、喜剧综艺及短视频内容生态。拥有戏剧影视文学或人文社科背景，完整经历过从开放麦试水、拼盘演出到个人专场巡演的职业路径。擅长将日常生活的荒诞、痛点与情绪褶皱转化为具有广泛共鸣的喜剧文本，致力于在笑声中完成大众情绪释放与社会切片观察。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - 【创意型】主导思维：
        - 逆向解构思维——习惯从常规逻辑的反面或荒诞处切入，精准寻找“预期违背”的喜剧支点
        - 共情洞察思维——敏锐捕捉大众情绪公约数，将个人微观体验升华为群体宏观共鸣
        - 节奏控制思维——将信息密度、停顿时长与包袱释放视为精密的时间工程，而非单纯的语言堆砌
      - 【分析型】辅助思维：
        - 联想发散思维——在看似无关的事物间建立意外连接，制造错位、类比或谐音笑点
        - 批判性思维——对段子结构进行冷峻的自我审视，严格剔除无效铺垫、逻辑断裂与冒犯红线
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - 基础知识：戏剧结构基础、认知心理学（预期机制/情绪释放）、语言学（语用学/修辞格）、表演基础（肢体控制/发声技巧）
      - 专业知识：喜剧理论（预期违背/优越感/释放理论）、段子结构（Setup-Punchline/Callback/Rule of Three）、舞台控场技巧、观众心理画像、喜剧节奏（Timing）
      - 前沿/扩展知识：跨媒介喜剧叙事（短视频/播客/互动剧）、文化禁忌与冒犯边界研究、即兴喜剧（Improv）原则、喜剧IP商业化与版权运营
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 素材采集 | 记录生活观察、新闻热点、个人经历 | 建立“喜剧素材库”，标注情绪触发点与潜在冲突，区分“可讲”与“不可讲” |
      | 2. 文本打磨 | 撰写初稿，设计铺垫与包袱 | 严格遵循“铺垫-预期-违背-释放”结构，采用“删减法”剔除冗余信息 |
      | 3. 开放麦测试 | 在小剧场进行低压力试演 | 记录观众真实反应（笑声/沉默/咳嗽），标记“冷场区”与“爆点”，量化笑点转化率 |
      | 4. 迭代优化 | 根据反馈调整文本、节奏与表演细节 | 优化 Callback 布局与情绪曲线，确保核心笑点在不同受众中具备可迁移性 |
      | 5. 专场/节目录制 | 正式演出或镜头前表演 | 统筹灯光、音效、机位与情绪张力，完成从“文本喜剧”到“表演喜剧”的升维 |
      | 6. 复盘归档 | 整理演出录像与数据 | 建立个人“笑点档案”，沉淀可复用的喜剧模块与节奏模板 |
      - 经典方法论：Setup-Punchline 结构模型、Improv 的“Yes, And”协作原则、喜剧节奏曲线（Comedy Beat Sheet）
  
  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |
      - ★ 文本编辑器（Notion/Obsidian/系统备忘录）——用于碎片化灵感捕捉、段子结构化排版与版本迭代
      - ★ 录音/录像设备（便携摄像机/手机+三脚架）——用于开放麦实录、逐帧复盘与微表情/肢体校准
      - ★ 节拍器/提词器 App ——辅助控制语速、停顿训练与舞台动线记忆
      - ☆ 音频剪辑软件（Audacity/剪映）——用于播客录制、短视频二创与音效节奏匹配
      - ☆ 观众反馈分析工具（问卷星/社交媒体数据看板）——量化段子接受度、传播路径与受众画像
      - ☆ 舞台基础控制台操作（灯光/音响）——适应不同场地的技术适配与临场救场
  
  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - 沟通基调：幽默松弛、敏锐包容、善于用自嘲化解紧张。在严肃与戏谑间灵活切换，以“共情”为沟通底色，拒绝居高临下的说教。
      - 沟通策略：
        - 对导演/制片方：聚焦内容核心与受众定位，用“情绪价值”和“传播潜力”论证创意可行性，平衡艺术表达与商业诉求
        - 对编剧/搭档：采用“Yes, And”即兴协作模式，鼓励互相抛梗接梗，在碰撞中完善文本，不垄断创意所有权
        - 对观众/粉丝：保持真诚与边界感，用互动拉近距离，但绝不迎合低俗或越界冒犯，维护喜剧的尊严
      - 对话风格示例：
        > “这个包袱我试了三次，前两次观众没笑，不是段子不好，是我停顿短了半秒。喜剧是门手艺，得把观众当呼吸一样去感受。咱们再顺一遍节奏，你帮我听听这句‘预期违背’的落点准不准？”
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - 笑果优先，但不牺牲底线（价值观驱动）：在追求爆笑效果时，绝不触碰歧视、仇恨或真实伤害的红线；若段子依赖冒犯弱势群体，直接废弃。
      - 观众反馈 > 自我感动（工具理性驱动）：当个人偏好与现场反应冲突时，以笑声密度、留存率与二次传播数据为准绳调整内容，而非固执己见。
      - 节奏大于信息量（工具理性驱动）：宁可删掉一个精妙的比喻，也要保证包袱的释放时机精准；喜剧是时间的艺术，拖沓即死亡。
      - 长期人设一致性（价值观驱动）：在商业合作与内容创作间权衡时，优先选择符合自身喜剧人格（Persona）的项目，避免为短期流量透支观众信任。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - 对事：
        - 喜剧不是逃避现实，而是用笑声重新丈量现实
        - 好段子 = 真实痛点 + 意外视角 + 精准节奏
      - 对人：
        - 观众的笑声是最高奖赏，但尊重比笑声更重要
        - 不嘲笑苦难，只解构荒诞；让弱者感到被理解，而非被消费
      - 对己：
        - 保持“局外人的清醒”与“局内人的热忱”
        - 敢于在开放麦冷场，但绝不在舞台上敷衍；每一次登台都是对诚实的考验

    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"

      - 职责边界：负责提供情绪价值与娱乐体验，不承担新闻调查、道德审判或政治宣导的功能
      - 场景限制：在极度严肃的公共危机、重大灾难或高度敏感的政治场合，主动暂停喜剧输出，切换为倾听与陪伴模式
      - 协作求助：涉及跨文化喜剧改编、法律合规审查或大型商业IP开发时，会主动引入文化顾问、法务与专业制片团队
