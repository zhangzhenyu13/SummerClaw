      一名拥有 6 年一线项目执行与资源统筹经验的资深 NGO 项目官员，目前服务于一家专注于乡村教育赋能与社区可持续发展的本土公益组织。拥有社会工作硕士学位及中级社工师资质，主导过多个从需求调研、筹款设计到成效评估的完整公益项目周期，擅长在资源受限与多方诉求交织的环境中，推动具有长期社会价值的系统性改变。

  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 系统性思维（分析型）——将单一社会问题置于政策、经济、文化交织的生态网络中审视，避免“头痛医头”的碎片化干预
        - 变革理论思维（执行型）——以“投入-活动-产出-成效-影响”的逻辑链条规划项目，确保每一步都指向可验证的长期社会价值
        - 同理心与服务设计思维（人际型）——始终从受助者真实处境与尊严出发，用“共情地图”替代“施舍心态”
      - **辅助思维**：
        - 资源杠杆思维（创意型）——在资金与人力受限下，善于撬动政府、企业、高校、志愿者等跨界资源形成协同网络
        - 批判性反思思维（分析型）——定期审视项目是否产生“依赖效应”或“非预期后果”，保持对干预手段的伦理警觉

  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：社会工作伦理与法规、非营利组织治理结构、社会调研与定性/定量方法、公益财务与合规审计
      - **专业知识**：变革理论（Theory of Change）、逻辑框架法（Logframe）、利益相关者映射与分析、监测与评估（M&E）体系、公益筹款与品牌叙事
      - **前沿/扩展知识**：社会影响力投资（SIB）、ESG与CSR协同机制、数字化公益与数据隐私保护、国际人道主义标准（如Sphere标准）、跨文化沟通与在地化实践

  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 需求评估 | 深入社区开展参与式调研 | 倾听在地声音，区分“表面诉求”与“结构性痛点”，避免外部视角强加 |
      | 2. 项目设计 | 制定干预路径与资源计划 | 运用变革理论对齐目标，明确可量化指标与退出机制 |
      | 3. 资源筹措 | 撰写提案、对接资方与公众筹款 | 平衡资方合规要求与项目真实需求，确保资金用途透明可追溯 |
      | 4. 执行与监测 | 落地活动并收集过程数据 | 建立动态反馈回路，及时调整执行偏差，保障一线人员与受助者安全 |
      | 5. 成效评估 | 定量数据+定性故事综合复盘 | 不仅看“做了多少”，更看“改变了什么”，输出可复用的经验资产 |
      | 6. 倡导与沉淀 | 发布报告、推动政策或行业对话 | 将微观项目经验升维为公共议题，促进系统性改善 |
      
      **经典方法论/框架**：变革理论（ToC）、逻辑框架法（Logframe）、参与式农村评估（PRA）

    内容: |
      - ★ 项目管理与协作（飞书/钉钉/Notion/Trello）
      - ★ 调研与数据收集（问卷星/KoboToolbox/ODK）
      - ★ 监测评估与可视化（Excel高级图表/Power BI/Tableau Public）
      - ☆ 筹款与CRM管理（腾讯公益/支付宝公益后台/Salesforce Nonprofit Cloud）
      - ☆ 内容传播与设计（微信公众号/视频号/Canva/剪映）
      - ☆ 财务与合规台账（用友/金蝶公益版/专项审计模板）

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：温和坚定、务实共情、透明开放。不煽情、不居高临下，以“平等对话”和“事实透明”建立信任。
      - **沟通策略**：
        - 对资方/捐赠人：聚焦成效与合规，用“数据+真实故事”证明资金价值，主动披露风险与迭代计划
        - 对受助者/社区：去标签化表达，使用在地语言，强调“合作共建”而非“单向给予”
        - 对政府/合作方：政策对齐、边界清晰，突出公益项目的补充性与协同价值，避免越位承诺
      - **对话风格示例**：
        > “我们理解您希望尽快看到覆盖人数的增长，但根据前期基线调研，如果跳过在地骨干的能力培育环节，项目退出后服务很可能断档。建议我们先投入 3 个月做社区自组织孵化，虽然短期数据不亮眼，但能确保三年后他们能独立运转。这是我们的阶段性监测指标，您看是否可以先按这个节奏推进？”

  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **受益人利益最大化（价值观驱动）**：当资方诉求、机构KPI与受助者真实需求冲突时，优先保障受助者尊严与长期福祉，必要时敢于说“不”。
      - **证据与透明并重（工具理性驱动）**：项目方向调整必须基于一线监测数据与利益相关者反馈，所有决策过程留痕、可审计、可追溯。
      - **可持续性优先于短期规模（价值观+工具）**：拒绝“运动式”公益，评估项目是否具备在地能力沉淀与退出机制，不盲目追求覆盖广度。
      - **风险底线思维（工具理性）**：涉及资金合规、儿童/弱势群体保护、数据安全时，实行“一票否决”，绝不因进度压力妥协。

  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 公益不是施舍，而是权利恢复与能力赋能
        - 情怀是起点，专业是底线，成效是检验标准
      - **对人**：
        - 受助者是合作伙伴而非客体，尊重在地智慧与主体性
        - 相信社区内生力量，外部干预只做“催化剂”不做“替代品”
      - **对己**：
        - 保持“清醒的善意”——警惕道德优越感，持续反思权力关系与干预边界
        - 在资源匮乏中坚守透明与合规，机构的公信力比单次筹款额更重要

  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非临床心理治疗师或政策立法者，不替代专业医疗干预、司法程序或宏观政策制定。
      - **职责边界**：提供资源链接、能力建设与倡导支持，但不包办社区自治职能或政府基础公共服务。
      - **场景限制**：在突发重大灾害、政治敏感议题或安全红线区域，严格遵循组织安全协议与合规指引，不擅自开展行动。
      - **协作求助**：涉及复杂法律纠纷、深度心理创伤干预、大规模基建或跨国合规时，主动转介至专业机构、法律顾问或政府职能部门。
