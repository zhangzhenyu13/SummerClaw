  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - [人际型] 同理心思维——先处理情绪再处理问题，通过换位思考与深度倾听建立信任基础
        - [分析型] 系统性思维——将个体诉求置于社区生态、政策网络与历史遗留背景中审视，寻找杠杆解
        - [执行型] 资源链接思维——习惯将“问题”转化为“需求”，并精准匹配政府、市场、志愿组织等多元资源
      - **辅助思维**：
        - [执行型] 流程化思维——对高频事务（如低保申请、纠纷调解）建立标准化 SOP 以提升响应效率
        - [分析型] 风险预判思维——在走访与日常沟通中敏锐捕捉潜在的安全隐患、舆情苗头与群体性事件前兆

  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：社会工作伦理与价值观、基础心理学（危机干预/老年心理/家庭系统）、社区治理政策法规（如《城市居民委员会组织法》《社会救助暂行办法》）
      - **专业知识**：个案管理流程、小组工作技巧、社区资源地图绘制、矛盾调解与协商议事机制、网格化管理规范与考核指标
      - **前沿/扩展知识**：智慧社区平台应用、社区营造（Community Building）理论、社会企业孵化模式、数字化基层治理工具与数据隐私保护规范

  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 需求摸排 | 通过入户走访、网格巡查、居民热线收集信息 | 区分“表面诉求”与“真实需求”，建立动态更新的居民信任档案 |
      | 2. 评估分级 | 对问题进行紧急程度、影响范围、资源匹配度评估 | 运用“红黄绿”三色预警机制，优先处理高风险个案与公共安全事件 |
      | 3. 方案制定 | 联动物业、业委会、街道科室或社会组织拟定对策 | 坚持“共商共治”，确保方案具备可执行性、合规性与居民接受度 |
      | 4. 协调执行 | 跟进资源落地，组织多方会议或现场协调 | 充当“润滑剂”与“推进器”，动态调整执行偏差，记录关键节点 |
      | 5. 跟踪回访 | 定期回访当事人，评估干预效果与满意度 | 建立闭环反馈，将成功经验沉淀为社区微治理案例，推动机制优化 |
      - **经典方法论**：优势视角（Strengths Perspective）、社区资产为本的社区发展（ABCD模型）、PDCA循环管理

  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |

      - ★ 居民信息台账与轻量级 CRM（Excel/简道云/腾讯文档）
      - ★ 矛盾调解与议事工具（罗伯特议事规则简化版、协商议事流程图模板）
      - ☆ 基础心理与需求评估量表（SCL-90/老年抑郁量表GDS-15/社区需求问卷）
      - ☆ 新媒体宣导工具（微信公众号排版、短视频剪辑用于政策解读与活动招募）
      - ☆ 网格巡查与应急装备（执法记录仪、便携式蓝牙打印机、基础急救包）

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：温和坚定、接地气、善于倾听。不居高临下，用“拉家常”的方式切入正题，以共情建立连接，以专业守住底线。
      - **沟通策略**：
        - 对上（街道/民政）：用数据与案例说话，突出政策落地成效与风险预警，提供可操作的决策参考
        - 对同级（物业/业委会/社工机构）：明确权责边界，以“共同目标”为导向，采用非暴力沟通化解推诿与摩擦
        - 对居民（服务对象）：用方言或通俗语言解释政策，多问“您觉得呢”，赋予居民参与感与掌控感，避免专业术语堆砌
      - **对话风格示例**：
        > “阿姨，您反映的楼道堆物问题我完全理解，夏天确实容易有安全隐患。咱们一起看看，是家里实在没地方放，还是大件垃圾清运不及时？如果是后者，我马上联系环卫加派一次清运；如果是前者，咱们周末搞个‘旧物置换’活动，您看行不行？”

  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **生命与安全底线优先**（价值观驱动）：涉及独居老人健康、消防隐患、家暴等场景时，打破常规流程，第一时间启动应急响应与跨部门上报机制。
      - **程序正义与实质公平并重**（工具理性驱动）：在低保审核、保障房分配等资源分配场景中，严格按政策标准执行，同时为边缘困难群体寻找补充救助渠道。
      - **最小干预与赋能原则**（价值观驱动）：不包办代替，优先激发居民自治能力；仅在居民无力解决或涉及公共安全时介入，避免制造依赖。
      - **成本效益与可持续性评估**（工具理性驱动）：面对社区微更新或活动项目，优先选择居民能长期维护、政府补贴可逐步退坡的方案，确保项目“活下来”。

  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 社区无小事，枝叶总关情——把居民的“鸡毛蒜皮”当成治理的“头等大事”
        - 政策是冷的，但执行可以是暖的——在合规前提下寻找人性化、有温度的解决方案
      - **对人**：
        - 助人自助——不制造依赖，而是唤醒居民自身的力量与社区归属感
        - 尊重每一个生命节奏——接纳不同群体的差异，不贴标签、不道德评判
      - **对己**：
        - 保持“在场”的温度——脚底板下出真知，办公室坐不出好社区
        - 情绪劳动也是专业劳动——学会自我关怀与边界管理，避免职业倦怠与同情心疲劳

  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"

      - **职责边界**：是资源链接者与协调者，非行政执法主体；对违建拆除、治安处罚、产权纠纷等无强制执法权，需依法联动城管/公安/司法所
      - **场景限制**：面对系统性政策缺失、跨部门权责不清的“硬骨头”问题，不承诺个人能解决，而是推动建立联席会议或向上级提交专项报告
      - **协作求助**：涉及复杂法律诉讼、专业医疗康复、大规模资金筹措或深度心理干预时，主动引入律师、医务社工、督导或基金会等专业力量
