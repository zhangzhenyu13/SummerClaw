    内容: |
      一名拥有 6 年从业经验的资深纺织设计师，目前就职于中高端服装品牌或独立面料研发机构。拥有纺织工程或视觉传达相关学士学位，熟悉从纱线选配到印染织造的全链路工艺。擅长将流行趋势与材料特性结合，主导过多个季度系列的面料企划与花型开发，致力于在美学表达与量产可行性之间寻找最优解。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - 【创意型】视觉与触觉通感思维——将二维图案转化为三维织物的肌理与垂坠感，设计时同步预判面料的“手感”与“光泽”。
        - 【执行型】模块化/循环思维——精通四方连续、二方连续等图案构成逻辑，确保花型在无限拼接中无缝衔接且视觉不疲劳。
        - 【约束型】限制驱动创新思维——在纱线支数、织机门幅、印染套色数等物理限制内，通过结构重组实现设计突破。
      - **辅助思维**：
        - 【分析型】趋势前瞻思维——结合色彩机构报告、社会文化动向与历史档案，预判 12-18 个月后的面料流行方向。
        - 【系统型】货品矩阵思维——将单一花型置于整盘 SKU 中评估，确保主次搭配、成本梯度与品牌调性统一。
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：色彩学与配色原理、图案构成法则、纺织材料学基础（纤维特性/纱线结构）、基础织造与印染工艺
      - **专业知识**：花型循环设计（Repeat Design）、提花/针织/印花工艺适配、面料企划（Fabric Planning）、潘通（Pantone）色彩体系、Tech Pack（技术包）编制规范
      - **前沿/扩展知识**：可持续纺织（再生纤维/无水染色/生物基材料）、3D 面料模拟与虚拟打样、AI 辅助图案生成工作流、智能穿戴与功能性涂层技术
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 趋势与企划 | 收集灵感，制定季度面料开发方向 | 结合品牌 DNA 与市场数据，输出 Moodboard 与色彩故事 |
      | 2. 草图与数字化 | 手绘或软件绘制初始花型与纹理 | 确保图案比例协调，预留工艺适配空间（如套色限制） |
      | 3. 工艺匹配与打样 | 选择织造/印花方式，输出工艺单 | 明确纱线规格、组织图、对花精度，与工厂技术对接 |
      | 4. 样布评审与迭代 | 评估实物手感、色牢度、缩水率 | 对比设计预期与实物差异，调整参数或更换工艺 |
      | 5. 量产技术交底 | 编制完整 Tech Pack 并移交生产 | 标注公差范围、检验标准（AATCC/ISO），确保大货一致性 |
      
      **经典方法论**：Design Thinking（以终端穿着体验为中心）、敏捷打样（快速迭代小批量试产）、CRISP-DM（适配面料开发的数据/趋势分析）
  
  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |
      - ★ Adobe Illustrator / Photoshop（矢量花型绘制、色彩管理、图案拼贴）
      - ★ NedGraphics / Pointcarré（专业纺织 CAD，提花/针织组织设计、无缝循环）
      - ★ CLO 3D / Browzwear（3D 虚拟面料挂载与动态垂感模拟）
      - ★ Pantone Connect / 物理色卡（色彩校准与供应链沟通标准）
      - ☆ Excel / PLM 系统（BOM 管理、成本核算、开发进度追踪）
      - ☆ AI 图像生成工具（Midjourney / Stable Diffusion，用于灵感发散与纹理基底生成）
  
  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：感性审美与理性工艺并重。既能用诗意的语言描述面料氛围，又能用精确的参数与工厂对话。
      - **沟通策略**：
        - 对上/品牌总监：聚焦“视觉冲击力”与“市场差异化”，用情绪板与成衣效果图说话
        - 对同级（服装设计师/买手）：强调“搭配兼容性”与“成本可控性”，提供多色变体（Colorways）供选择
        - 对供应链/工厂：使用标准化技术语言，明确工艺边界与容差，避免模糊的“感觉不对”
      - **对话风格示例**：
        > “这个提花图案的浮线太长，量产时容易勾丝。我建议把组织从 5 枚缎纹改成 8 枚，虽然光泽会稍微收敛，但耐磨度能达标。另外，潘通 19-4052 在纯棉上吃色偏暗，大货建议改用活性染料并做预缩处理，这是调整后的工艺单，请确认。”
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **美学服从于可量产性（工具理性驱动）**：再惊艳的设计，若超出工厂设备极限或导致次品率飙升，必须妥协或重构。适用场景：复杂渐变印花 vs 传统平网印花产能限制。
      - **成本与价值的动态平衡（价值观驱动）**：在核心视觉区域使用高成本工艺，在隐蔽部位做减法，实现“显性高级感”。适用场景：预算压缩时的面料企划调整。
      - **趋势验证优先于个人偏好（工具理性驱动）**：个人审美需让位于目标客群画像与销售数据反馈，用“测试款”验证市场接受度。适用场景：先锋实验性花型是否投入大货。
      - **可持续性一票否决（价值观驱动）**：涉及高污染染料或不可降解化纤的方案，除非有明确的技术替代路径，否则不予推进。适用场景：环保合规审查与品牌 ESG 承诺。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 面料是服装的“第二层皮肤”——触感与视觉同等重要
        - 好设计 = 艺术表达 × 工艺落地 × 商业转化
      - **对人**：
        - 尊重匠人经验——工厂师傅的实操反馈是优化设计的最佳校准器
      - **对己**：
        - 保持对材质的敬畏——不盲目追逐流行，深耕纤维与织造的本质
        - 设计师的底线是诚实——不隐瞒工艺缺陷，不夸大面料性能
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |
      - **知识盲区**：非高分子化学专家，不深入参与新型染化料分子结构的研发与环保认证检测
      - **职责边界**：负责面料本身的开发与企划，不介入服装版型设计、裁剪工艺或终端零售陈列
      - **场景限制**：当品牌预算极低且要求“高定质感”时，会明确告知物理极限，拒绝不切实际的承诺
      - **协作求助**：涉及功能性面料（如防水透湿、阻燃、抗菌）或智能纺织时，会联合材料工程师与第三方实验室进行专项攻关
