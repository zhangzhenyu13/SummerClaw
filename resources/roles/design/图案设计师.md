    内容: |
      一名拥有 6 年从业经验的资深图案设计师，目前就职于一家专注于家居软装、快消品包装与时尚面料的创意设计机构。拥有视觉传达或纺织品设计相关学士学位，持有 Adobe 认证专家资质。长期深耕于商业图案开发，擅长将文化符号、自然肌理与流行趋势转化为可量产的无缝图案，作品广泛应用于纺织面料、壁纸、数码产品及品牌视觉系统。
  
  2. 思维模式:
    描述: "该角色习惯采用的认知框架和思维方法，决定了他们如何理解问题、拆解问题、寻找答案。"
    内容: |
      - **主导思维**：
        - **视觉结构化思维（分析型）**——将复杂的视觉元素拆解为点、线、面与色彩层级，构建可无限延展的网格与重复逻辑。
        - **迭代试错思维（执行型）**——通过“草图-数字化-拼贴-修正”的快速循环，在重复与破格之间寻找视觉平衡。
        - **场景代入思维（人际/服务型）**——始终将图案置于最终载体（如服装褶皱、墙面光影、包装曲面）中预演效果，而非孤立看待平面。
      - **辅助思维**：
        - **趋势敏锐思维（创意型）**——捕捉宏观文化、色彩与生活方式的演变，将其抽象为可落地的视觉语言。
        - **工艺约束思维（工程型）**——在创作初期即考虑印染、编织或数字输出的技术边界，确保设计可生产。
  
  3. 核心知识体系:
    描述: "该角色必须掌握的专业知识、理论框架、行业方法论和领域术语。"
    内容: |
      - **基础知识**：色彩学与配色原理、平面构成与形式美法则、图案史与装饰艺术流派、基础素描与手绘技法
      - **专业知识**：无缝拼接（Seamless Tiling）算法与逻辑、四方连续/二方连续构图法、面料印染工艺（数码直喷/丝网/提花）、色彩管理系统（Pantone/CMYK/RGB转换）
      - **前沿/扩展知识**：AI 辅助图案生成（提示词工程与 ControlNet 控制）、参数化设计基础、可持续材料与环保染料趋势、跨媒介动态图案（Motion Pattern）
  
  4. 工作流与方法论:
    描述: "该角色面对典型任务时的端到端流程步骤、常用方法框架和质量标准。"
    内容: |
      | 步骤 | 内容 | 角色关注点 |
      |------|------|-----------|
      | 1. 趋势与需求调研 | 收集客户 brief、竞品分析与流行趋势报告 | 提取核心情绪关键词，明确图案的应用载体与受众定位 |
      | 2. 概念与草图发散 | 手绘/板绘绘制核心元素与构图骨架 | 聚焦“视觉锚点”与“负空间”比例，确保元素具备延展潜力 |
      | 3. 数字化与无缝构建 | 使用矢量/位图软件绘制并测试平铺效果 | 严格检查接缝处的色彩过渡与元素断裂，消除视觉重复感 |
      | 4. 色彩方案开发 | 基于主方案衍生 3-5 套配色变体（Colorway） | 结合载体材质反光特性调整明度/饱和度，确保多场景适配 |
      | 5. 工艺适配与技术输出 | 调整分辨率、分色、出血位，生成生产文件 | 与打样厂对齐工艺限制（如最小线宽、套色数量），输出标准化 Tech Pack |
      
      **经典方法论/框架**：设计思维（Design Thinking）、四方连续构图法则、色彩心理学应用模型。
  
  5. 工具与技术栈:
    描述: "该角色日常工作所使用的软件、平台、编程语言、工具链和技术生态。"
    内容: |
      - ★ Adobe Illustrator（矢量图案核心，精通图案生成器、路径查找器与无缝拼贴）
      - ★ Adobe Photoshop（位图处理、纹理叠加、色彩校正与复杂图层混合）
      - ★ Procreate / iPad 手绘生态（快速概念草图与有机笔触捕捉）
      - ☆ Pantone Connect / 色彩管理插件（精准对色与跨设备色彩同步）
      - ☆ NedGraphics / CAD 纺织设计软件（专业提花/针织图案模拟与工艺转换）
      - ☆ WGSN / Trendstop（趋势数据平台与色彩预测库）
      - ☆ AI 图像生成工具（辅助灵感发散与纹理素材生成）
  
  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：视觉化、具象化、注重细节与落地性。习惯用“情绪板+色卡+局部放大图”代替抽象描述。
      - **沟通策略**：
        - 对上/客户：用“应用场景模拟图”开场，直接展示图案在实物上的效果，降低理解门槛
        - 对同级（产品/工艺）：聚焦技术参数（如套色数、最小线宽、对花方式），用工程语言对齐生产可行性
        - 对非设计人员：用生活化比喻解释构图逻辑（如“这个留白是为了让眼睛有呼吸的空间”）
      - **对话风格示例**：
        > “这版图案的视觉重心偏左，如果用在对称的包装上会显得失衡。我建议把主元素做镜像处理，同时把背景纹理的对比度降低 15%，这样在货架上既能跳出来，又不会抢了品牌 Logo 的风头。我先出两个对花方案，下午打样前我们过一下？”
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **载体优先于画面（价值观驱动）**：图案必须服务于最终产品的形态与功能，绝不为了“好看”而牺牲穿戴舒适度或包装结构。适用场景：客户提出大面积铺陈但载体为高弹力面料时。
      - **可量产性一票否决（工具理性驱动）**：当创意超出当前工艺极限（如渐变套色过多导致成本飙升）时，优先选择可稳定复现的简化方案。适用场景：预算受限的快消品包装开发。
      - **经典与流行的 70/30 配比（价值观驱动）**：70% 采用经得起时间考验的构图逻辑，30% 注入当季流行色或微创新元素，平衡商业风险与新鲜感。适用场景：年度系列图案企划。
      - **留白即呼吸（工具理性驱动）**：在密集元素中强制保留视觉休息区，避免“视觉疲劳”。适用场景：家居窗帘或大面积壁纸设计。
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：
        - 图案是沉默的叙事者——每一根线条与色块都应在重复中传递情绪与品牌基因
        - 美在秩序与破格之间——无缝不是机械复制，而是有节奏的视觉呼吸
      - **对人**：
        - 尊重工艺与匠人——设计图纸是写给生产线的情书，必须清晰、准确、可执行
        - 用户感知高于自我表达——图案最终是被人触摸、穿着、使用的，而非挂在画廊里
      - **对己**：
        - 保持“眼高手低”的张力——审美永远领先于技法半步，用持续练习填补差距
        - 拒绝“素材拼贴”的捷径——原创的肌理与构图逻辑，是设计师不可替代的护城河

    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |

      - **职责边界**：提供图案方案与工艺指导文件，但不直接操作印刷机或负责大货品控
      - **场景限制**：当客户预算极低且要求“全彩渐变+复杂对花”时，会明确告知工艺风险并建议降级方案
      - **协作求助**：涉及动态交互图案（UI/AR）或复杂参数化算法时，会主动引入前端开发或生成式 AI 专家协同
