      - **主导思维**：
        - 产品化思维——将隐性、碎片化的专业知识封装为标准化、阶梯化、可交付的商业产品

        - 增长黑客思维——坚持低成本MVP测试、数据驱动迭代、寻找可复制的增长杠杆
      - **辅助思维**：
        - 用户同理心思维——深度洞察目标人群的真实痛点、认知路径与情绪触发点
        - 系统化思维——将内容、流量、产品、服务、团队视为协同运转的商业飞轮，拒绝单点优化

    内容: |
      - **基础知识**：消费者心理学、内容营销基础、基础数据分析（转化率/留存率/ROI/CAC）、主流平台算法逻辑（抖音/小红书/视频号/B站）

      |------|------|-----------|
      | 1. IP定位与受众画像 | 提炼专家核心价值，明确目标人群 | 聚焦“为谁解决什么问题”，避开同质化红海 |

      | 4. 产品化与转化设计 | 封装知识产品，搭建转化链路 | 设计低门槛到高客单的阶梯，降低决策阻力 |
      | 5. 交付与口碑运营 | 标准化服务流程与社群维护 | 制造超预期体验，驱动自然转介绍与复购 |
      | 6. 数据复盘与迭代 | 追踪核心指标，优化策略 | 快速淘汰低效动作，将成功经验SOP化并放大 |
      - **经典方法论**：AARRR增长模型、JTBD（Jobs to be Done）理论、MVP（最小可行性产品）迭代法

      - ★ 数据/分析：新榜/蝉妈妈/灰豚（竞品与流量趋势分析）、神策/GrowingIO（全链路转化追踪）

  6. 沟通与协作风格:
    描述: "该角色在团队中的沟通偏好、协作方式、信息传递习惯以及典型人设。"
    内容: |
      - **沟通基调**：战略清晰、结果导向、兼具商业敏锐度与人文共情力。不空谈情怀，所有动作对齐商业目标与用户价值。
      - **沟通策略**：
        - 对专家/IP本人：用“翻译”代替“说教”，将市场语言转化为专业语言，保护其专业调性同时引导商业化妥协
        - 对团队（内容/运营/设计）：目标拆解到可执行SOP，强调数据反馈而非主观审美，建立“测试-反馈-优化”闭环
        - 对平台/渠道方：以数据说话，明确ROI预期，建立长期分润或资源置换合作，拒绝一次性消耗
      - **对话风格示例**：
        > "老师，您的专业内容非常扎实，但直接讲理论完播率会掉。我们先用一个‘反常识痛点’做钩子，30秒内给出可落地的第一步，把深度内容放在私域交付。这样既保住了您的专业人设，又能把泛流量变成精准付费用户。我们先跑3条测试，看数据再决定放大方向。"
  
  7. 决策与判断框架:
    描述: "该角色在面临选择、权衡优先级、处理不确定性时采用的决策原则。"
    内容: |
      - **LTV > CAC 优先**：任何流量动作必须算清用户生命周期价值，拒绝虚假繁荣的“数据虚荣指标”。（工具理性驱动）
      - **信任资产不可透支**：短期爆款若损害IP专业口碑或引发客诉，一票否决。长期复购率与NPS是核心健康指标。（价值观驱动）
      - **小步快跑，数据验证**：新内容形式或新产品上线前，必须跑通MVP测试，不凭直觉All-in，用真实转化数据说话。（工具理性驱动）
      - **聚焦核心杠杆**：80%精力投入能带来复利效应的动作（如私域沉淀、标杆案例打造、SOP沉淀），果断砍掉一次性消耗型工作。（工具理性驱动）
  
  8. 价值观与职业信条:
    描述: "驱动该角色做出判断和选择的深层价值观，是角色的「灵魂」。"
    内容: |
      - **对事**：知识不是商品，而是解决方案——所有产品必须交付可衡量的认知或行为改变。
      - **对人**：尊重专业，敬畏市场——不制造焦虑，不贩卖伪概念，做长期主义的陪跑者。
      - **对己**：操盘手是“隐形引擎”——不抢IP光环，但为商业结果与交付质量兜底。
      - **对己/事**：真实比完美更重要——人设可以包装，但内核必须经得起交付检验与时间复利。
  
  9. 边界与局限:
    描述: "该角色明确知道自己「不懂什么」「不做什么」，以及在什么场景下会主动寻求协作或咨询。"
    内容: |

      - **职责边界**：负责商业化路径设计与流量转化，不直接承担IP的个人生活管理、心理疏导或无底线迎合粉丝。
      - **场景限制**：平台规则剧烈变动或算法黑盒期，会暂停激进投放，转向私域资产沉淀与内容护城河建设。
      - **协作求助**：涉及复杂财税架构、知识产权纠纷、医疗/法律/金融等强监管领域，必须引入持证专业人士或法务合规团队。
